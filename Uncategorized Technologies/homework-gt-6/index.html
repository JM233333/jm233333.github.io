6.26

6.26. 设 G1,G2,...,GkG_1, G_2, ..., G_kG1​,G2​,...,Gk​ 是图 GGG 的所有块，证明 χ(G)=max⁡{χ(Gi)∣ i∈[1, k]}\chi(G) = \max\{\chi(G_i) \mid \ i \in [1,\ k]\}χ(G)=max{χ(Gi​)∣ i∈[1, k]} 。

显然对于 ∀G′⊆G\forall G' \subseteq G∀G′⊆G 有 χ(G)≥χ(G′)\chi(G) \ge \chi(G')χ(G)≥χ(G′) ，所以我们只需证明 χ(G)≤max⁡{χ(Gi)∣ i∈[1, k]}\chi(G) \le \max\{\chi(G_i) \mid \ i \in [1,\ k]\}χ(G)≤max{χ(Gi​)∣ i∈[1, k]} 即可。

若 GGG 是非连通图，则显然有 χ(G)=max⁡{χ(G′)∣G′⊆G}\chi(G) = \max\{\chi(G') \mid G' \subseteq G\}χ(G)=max{χ(G′)∣G′⊆G} ，因此我们只需考虑 GGG 是连通图的情况即可。

接下来我们证明，对于任意一个连通图 GGG ，总能构造出一种满足条件的点染色方案，即可证明原命题。

我们从 GGG 的任意一个块（不妨记为 G0G_0G0​ ）出发，首先以 G0G_0G0​ 的任意一种正常 χ(G0)\chi(G_0)χ(G0​) 染色方案为 G0G_0G0​ 染色。

由块的性质可知，对于 GGG 的任意一个块 Gi (i≠0)G_i\ (i \ne 0)Gi​ (i​=0) ，若 GiG_iGi​ 与 G0G_0G0​ 相邻，则 GiG_iGi​ 和 G0G_0G0​ 有且仅有一个公共点，不妨记为 v0, iv_{0,\ i}v0, i​ 。

考虑 GiG_iGi​ 的任意一种正常 χ(Gi)\chi(G_i)χ(Gi​) 染色方案，如果此方案中对点 vvv 的染色 c′c'c′ 与 v0, iv_{0,\ i}v0, i​ 的已有染色 ccc 发生冲突，由于该点是唯一的公共点，即唯一的颜色冲突点，我们可对 GiG_iGi​ 的此染色方案进行无冲突的映射，例如可将所有染色为 ccc 的点改用 c′c'c′ 、将所有染色为 c′c'c′ 的点改用 ccc 。

接下来，我们将 G0∪GiG_0 \cup G_iG0​∪Gi​ 视为一个整体，然后重复上述过程，即寻找下一个相邻的块。注意到如下性质：

  • 因为 GGG 是连通的，所以此构造方案一定会遍历到 GGG 中的所有块；

  • 不可能在遍历到一个块 GkG_kGk​ 时发现 GkG_kGk​ 同时与已染色的块 GiG_iGi​ 和 GjG_jGj​ 存在交点，否则 Gi∪Gj∪GkG_i \cup G_j \cup G_kGi​∪Gj​∪Gk​ 构成一个 2−2-2− 连通图，与块的极大性质相矛盾。

综上，我们总能找到一种方案，使得对每个块 GGG 的染色方案之间互不冲突，因此 χ(G)≤max⁡{χ(Gi)∣ i∈[1, k]}\chi(G) \le \max\{\chi(G_i) \mid \ i \in [1,\ k]\}χ(G)≤max{χ(Gi​)∣ i∈[1, k]} ，原命题得证。

6.35

6.35. 证明：(1) 若 GGG 中的任意两个奇环都有公共点，则 χ(G)≤5\chi(G) \le 5χ(G)≤5 ；(2) 若 GGG 中的任意两个奇环都没有公共点，则 χ(G)≤3\chi(G) \le 3χ(G)≤3 ；(3) 若 GGG 中的奇环个数不超过 222 ，则 χ(G)≤3\chi(G) \le 3χ(G)≤3 。

证明 (1) ：

设 CCC 是 GGG 中的一个奇环，则 χ(C)=3\chi(C) = 3χ(C)=3 。因为 GGG 中的任意两个奇环都有公共点，所以 G−CG - CG−C 中不含奇环，所以 G−CG - CG−C 是一个二分图（可能为空图），进而有 χ(G−C)≤2\chi(G - C) \le 2χ(G−C)≤2 。

若我们在 CCC 和 G−CG - CG−C 中使用无交集的颜色集合进行点染色，则相当于用 χ(C)+χ(G−C)\chi(C) + \chi(G - C)χ(C)+χ(G−C) 种颜色为 GGG 进行点染色。显然这是一个正常点染色方案，所以 χ(G)≤χ(C)+χ(G−C)≤5\chi(G) \le \chi(C) + \chi(G - C) \le 5χ(G)≤χ(C)+χ(G−C)≤5 。

证明 (2) ：

设 GGG 中的所有奇环为 C1,C2,...,CkC_1, C_2, ..., C_kC1​,C2​,...,Ck​ 。

我们首先证明：设 CiC_iCi​ 和 CjC_jCj​ 是 GGG 中的任意两个奇环，若 GGG 中的任意两个奇环都没有公共点，则 GGG 上所有从 CiC_iCi​ 到 CjC_jCj​ 的简单路径都共享相同的端点。为简单起见，我们设这里所言的路径不会经过 CiC_iCi​ 或 CjC_jCj​ 上的多个点，因为这对结论没有影响。

设 vi, vi′∈Civ_i,\ v_i' \in C_ivi​, vi′​∈Ci​ , vj, vj′∈Cjv_j,\ v_j' \in C_jvj​, vj′​∈Cj​ ，假设存在两条路径 vi→R→vjv_i \to R \to v_jvi​→R→vj​ 和 vi′→R′→vj′v_i' \to R' \to v_j'vi′​→R′→vj′​ 满足 vi≠vi′∨vj≠vj′v_i \ne v_i' \lor v_j \ne v_j'vi​​=vi′​∨vj​​=vj′​ 。不妨设 vi≠vi′∧vj=vj′v_i \ne v_i' \land v_j = v_j'vi​​=vi′​∧vj​=vj′​ （若 vi≠vi′∧vj≠vj′v_i \ne v_i' \land v_j \ne v_j'vi​​=vi′​∧vj​​=vj′​ ，则可将 CjC_jCj​ 上从 vj′v_j'vj′​ 到 vjv_jvj​ 的路径纳入 RRR 中，在接下来的证明中可以看到这对结论没有影响）。

而在奇环 CiC_iCi​ 上一定存在且仅存在两条从 viv_ivi​ 到 vi′v_i'vi′​ 的路径，且其长度分别为奇数和偶数，记为 LoddL_{odd}Lodd​ 和 LevenL_{even}Leven​ 。此时不难发现：
