3. λ−\lambda-λ− 演算

语法

λ−\lambda-λ− 演算的字母表：

  • 变元集合 ∇={v,v′,v′′,...}\nabla = \{v, v', v'', ...\}∇={v,v′,v′′,...} ，该集合无穷。

  • 抽象算子 λ\lambdaλ 。

  • 括号 ((( 和 ))) 。

λ−\lambda-λ− 项的集合 Λ\LambdaΛ 归纳定义为满足以下条件的最小集合：

  • x∈∇→x∈Λx \in \nabla \rightarrow x \in \Lambdax∈∇→x∈Λ

  • M, N∈Λ→(MN)∈ΛM,\ N \in \Lambda \rightarrow (MN) \in \LambdaM, N∈Λ→(MN)∈Λ

  • x∈∇, M∈Λ→(λxM)∈Λx \in \nabla,\ M \in \Lambda \rightarrow (\lambda{xM}) \in \Lambdax∈∇, M∈Λ→(λxM)∈Λ

设 M∈ΛM \in \LambdaM∈Λ ，定义 MMM 的长度 ρ(M)ρ(M)ρ(M) 为 MMM 中变元出现的次数，即：

  • ρ(x)=1, (x∈∇)ρ(x) = 1,\ (x \in \nabla)ρ(x)=1, (x∈∇)

  • ρ(MN)=ρ(M)+ρ(N), (M, N∈Λ)ρ(MN) = ρ(M) + ρ(N),\ (M,\ N \in \Lambda)ρ(MN)=ρ(M)+ρ(N), (M, N∈Λ)

  • ρ(λx.M)=ρ(M)+1, (x∈∇, M∈Λ)ρ(\lambda{x.M}) = ρ(M) + 1,\ (x \in \nabla,\ M \in \Lambda)ρ(λx.M)=ρ(M)+1, (x∈∇, M∈Λ)

我们说对 λ−\lambda-λ− 项 MMM 的结构作归纳，就是指对 MMM 的长度 ρ(M)ρ(M)ρ(M) 作归纳。

子项集合。

设 M∈ΛM \in \LambdaM∈Λ ，归纳定义 MMM 的子项集合 Sub(M)Sub(M)Sub(M) 如下：

  • Sub(x)={x}Sub(x) = \{x\}Sub(x)={x}

  • Sub(N1, N2)=Sub(N1)∪Sub(N2)∪{N1, N2}Sub(N_1,\ N_2) = Sub(N_1) \cup Sub(N_2) \cup \{N_1,\ N_2\}Sub(N1​, N2​)=Sub(N1​)∪Sub(N2​)∪{N1​, N2​}

  • Sub(λx.N)=Sub(N)∪{λx.N}Sub(\lambda{x.N}) = Sub(N) \cup \{\lambda{x.N}\}Sub(λx.N)=Sub(N)∪{λx.N}

若 N∈Sub(M)N \in Sub(M)N∈Sub(M) ，则称 NNN 为 MMM 的子项。

自由变元集合。

设 M∈ΛM \in \LambdaM∈Λ ，归纳定义 MMM 的自由变元集合 FV(M)FV(M)FV(M) 如下：

  • FV(x)={x}FV(x) = \{x\}FV(x)={x}

  • FV(N1N2)=FV(N1)∪FV(N2)FV(N_1N_2) = FV(N_1) \cup FV(N_2)FV(N1​N2​)=FV(N1​)∪FV(N2​)

  • FV(λx.N)=FV(N)−{x}FV(\lambda{x.N}) = FV(N) - \{x\}FV(λx.N)=FV(N)−{x}

对于出现于 MMM 中的 xxx ，若 x∈FV(M)x \in FV(M)x∈FV(M) ，则称 xxx 是 自由变元 (free variable) ；若 x∉FV(M)x \notin FV(M)x∈/​FV(M) ，则称 xxx 是 约束变元 (bounded variable) 。

若 FV(M)=∅FV(M) = \emptysetFV(M)=∅ ，则称 MMM 为 闭 λ−\lambda-λ− 项 (closed lambda-term) 或 组合子 (combinator) 。用 Λ∘\Lambda^{\circ}Λ∘ 表示全体闭 λ−\lambda-λ− 项的集合。

变元改名。

变元约定：若 M1,M2,...,MnM_1, M_2, ..., M_nM1​,M2​,...,Mn​ 出现在某个数学表述过程中，则在这些项中所有的约束变元不同于其中的自由变元。

我们用 M[x:=N]M[x := N]M[x:=N] 表示将 MMM 中所有自由出现的 xxx 用 NNN 替代后所得到的结果，该结果被归纳定义如下：

  • x[x:=N]≡Nx[x := N] \equiv Nx[x:=N]≡N

  • y[x:=N]≡y, (x≢y)y[x := N] \equiv y,\ (x \not\equiv y)y[x:=N]≡y, (x​≡y)

  • (M1M2)[x:=N]≡(M1[x:=N])(M2[x:=N])(M_1M_2)[x := N] \equiv (M_1[x := N])(M_2[x := N])(M1​M2​)[x:=N]≡(M1​[x:=N])(M2​[x:=N])

  • (λy.M)[x:=N]≡λy.M[x:=N](λy.M)[x := N] \equiv \lambda{y.M[x := N]}(λy.M)[x:=N]≡λy.M[x:=N] ，注意这一条里不需要加上条件 y≢xy \not\equiv xy​≡x 且 y∉FV(N)y \notin FV(N)y∈/​FV(N) ，因为之前的变元约定保证了这些条件的成立。

引理 3.2 ：

  • M[x:=x]≡MM[x := x] \equiv MM[x:=x]≡M

  • x∉FV(M)→M[x:=N]≡Mx \notin FV(M) \rightarrow M[x := N] \equiv Mx∈/​FV(M)→M[x:=N]≡M

  • x∈FV(M)→FV(M[x:=N])=FV(N)∪(FV(M)−{x})x \in FV(M) \rightarrow FV(M[x := N]) = FV(N) \cup (FV(M) − \{x\})x∈FV(M)→FV(M[x:=N])=FV(N)∪(FV(M)−{x})

  • ρ(M[x:=y])=ρ(M)ρ(M[x := y]) = ρ(M)ρ(M[x:=y])=ρ(M)

替换引理：若 x≢yx \not\equiv yx​≡y 且 x∉FV(L)x \notin FV(L)x∈/​FV(L) ，则 M[x:=N][y:=L]≡(M[y:=L])[x:=N[y:=L]]M[x := N][y := L] \equiv (M[y := L])[x := N[y := L]]M[x:=N][y:=L]≡(M[y:=L])[x:=N[y:=L]] 。

转换

(λx.M)N=M[x:=N](\lambda{x.M})N = M[x := N](λx.M)N=M[x:=N]

(λx1x2...xn.M)Z1Z2...Zn=M[x1=Z1][x2=Z2]...[xn=Zn](\lambda{x_1x_2...x_n.M})Z_1Z_2...Z_n = M[x_1 = Z_1][x_2 = Z_2]...[x_n = Z_n](λx1​x2​...xn​.M)Z1​Z2​...Zn​=M[x1​=Z1​][x2​=Z2​]...[xn​=Zn​]

设 M∈ΛM \in \LambdaM∈Λ 且 FV(M)={x1,x2,...,xn}FV(M) = \{x_1, x_2, ..., x_n\}FV(M)={x1​,x2​,...,xn​} ，则 ∃F∈Λ∘, Fx1x2...xn=M\exists F \in \Lambda^{\circ},\ Fx_1x_2...x_n = M∃F∈Λ∘, Fx1​x2​...xn​=M 。取 F=λx1x2...xn.MF = \lambda{x_1x_2...x_n.M}F=λx1​x2​...xn​.M 即可。

定义如下标准组合子：

  • I≡λx.x\bm{I} \equiv \lambda{x.x}I≡λx.x

  • K≡λxy.x\bm{K} \equiv \lambda{xy.x}K≡λxy.x

  • K∗≡λxy.y\bm{K^*} \equiv \lambda{xy.y}K∗≡λxy.y

  • S≡λxyz.xz(yz)\bm{S} \equiv \lambda{xyz.xz(yz)}S≡λxyz.xz(yz)

由定义可得：

  • IM=M\bm{I}M = MIM=M

  • KMN=M\bm{K}MN = MKMN=M

  • K∗MN=N\bm{K^*}MN = NK∗MN=N

  • SMNL=ML(NL)\bm{S}MNL = ML(NL)SMNL=ML(NL)

ML=NL↛M=NML = NL \not\rightarrow M = NML=NL​→M=N

(ext) Mx=Nx→M=N, (x∉FV(MN))Mx = Nx \to M = N,\ (x \notin FV(MN))Mx=Nx→M=N, (x∈/​FV(MN))

(μ\muμ) λx.Mx=M, (x∉FV(M))\lambda{x.Mx} = M,\ (x \notin FV(M))λx.Mx=M, (x∈/​FV(M)) 表示了所谓 μ−\mu-μ− 转换。

规约

不动点定理

对于 ∀F∈Λ\forall F \in \Lambda∀F∈Λ 有 ∃Z∈Λ, FZ=βZ\exists Z \in \Lambda,\ FZ =_{\beta} Z∃Z∈Λ, FZ=β​Z 。

存在组合子 Y∈Λ∘\bm{Y} \in \Lambda^{\circ}Y∈Λ∘ ，使得 ∀F∈Λ, F(YF)=βYF\forall F \in \Lambda,\ F(\bm{Y}F) =_{\beta} \bm{Y}F∀F∈Λ, F(YF)=β​YF 。

对于 ∀F∈Λ\forall F \in \Lambda∀F∈Λ 有 ∃Z∈Λ, ZF=βZ\exists Z \in \Lambda,\ ZF =_{\beta} Z∃Z∈Λ, ZF=β​Z 。

ZF=βZ←F=βλx.Z←F=β(λfx.f)Z←F≡Y(λfx.f)\begin{aligned} ZF =_{\beta} Z &\leftarrow F =_{\beta} \lambda{x.Z}\\ &\leftarrow F =_{\beta} (\lambda{fx.f})Z\\ &\leftarrow F \equiv \bm{Y}(\lambda{fx.f}) \end{aligned}ZF=β​Z​←F=β​λx.Z←F=β​(λfx.f)Z←F≡Y(λfx.f)​

令 A≡λxy.y(xxy)A \equiv \lambda{xy.y(xxy)}A≡λxy.y(xxy) 且 Θ≡AA\Theta \equiv AAΘ≡AA ，则对于 ∀F∈Λ\forall F \in \Lambda∀F∈Λ 有 ΘF↠βF(ΘF)\Theta{F} \twoheadrightarrow_{\beta} F(\Theta{F})ΘF↠β​F(ΘF) 。

追加定义：

  • Uin≡λx1x2...xn.xi (n≥1, i∈[1, n])U_i^n \equiv \lambda{x_1x_2...x_n.x_i}\ (n \ge 1,\ i \in [1,\ n])Uin​≡λx1​x2​...xn​.xi​ (n≥1, i∈[1, n])

  • [M1,M2,...,Mn]≡λz.zM1M2...Mn (n≥1, Mi∈Λ)[M_1, M_2, ..., M_n] \equiv \lambda{z.zM_1M_2...M_n}\ (n \ge 1,\ M_i \in \Lambda)[M1​,M2​,...,Mn​]≡λz.zM1​M2​...Mn​ (n≥1, Mi​∈Λ) ，其中 zzz 为不出现在 M1,M2,...,MnM_1, M_2, ..., M_nM1​,M2​,...,Mn​ 中的新变元；

  • (M)in≡MUin (n≥1, i∈[1, n])(M)_i^n \equiv MU_i^n\ (n \ge 1,\ i \in [1,\ n])(M)in​≡MUin​ (n≥1, i∈[1, n])

  • ([M1,M2,...,Mn])in↠βMi (n≥1, i∈[1, n])([M_1, M_2, ..., M_n])_i^n \twoheadrightarrow_{\beta} M_i\ (n \ge 1,\ i \in [1,\ n])([M1​,M2​,...,Mn​])in​↠β​Mi​ (n≥1, i∈[1, n])

与递归论对应的结果

λ−\lambda-λ− 项的 Godel 编码：设 ♯:Λ→N\sharp : \Lambda \to \bm{N}♯:Λ→N ，若对于 ∀M∈Λ\forall M \in \Lambda∀M∈Λ 都有唯一的自然数 ♯M\sharp{M}♯M 与之对应，称 ♯M\sharp{M}♯M 为 MMM 的编码。定义 ♯\sharp♯ 的方法很多，例如：

  • ♯(v(n))=[0, n]\sharp{(v^{(n)})} = [0,\ n]♯(v(n))=[0, n]

  • ♯(MN)=[1, [♯M, ♯N]]\sharp{(MN)} = [1,\ [\sharp{M},\ \sharp{N}]]♯(MN)=[1, [♯M, ♯N]]

  • ♯(λx.M)=[2, [♯x, ♯M]]\sharp{(\lambda{x.M})} = [2,\ [\sharp{x},\ \sharp{M}]]♯(λx.M)=[2, [♯x, ♯M]]
