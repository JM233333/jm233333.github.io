LeetCode 0049 - Group Anagrams

Given an array of strings, group anagrams together.

Example:

Note:

  • All inputs will be in lowercase.

  • The order of your output does not matter.

题意

给定 nnn 个字符串，称一个字符串的本质串为将其内部字母排序后得到的字符串，要求将所有本质相同的字符串放在一个 vector 内，返回所有的 vector 。

题解

显然每一个本质串都对应了一个 vector ，我们用 unordered_map 实现该映射，遍历所有字符串并求出其本质串，根据映射放入对应的 vector 中，最后将所有的 vector 取出即可。
