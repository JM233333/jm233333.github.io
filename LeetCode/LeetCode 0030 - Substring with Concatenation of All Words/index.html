Difficulty Time Complexity Limit Extra-Memory Complexity Limit
Hard O(nL)O(nL)O(nL) −-−

You are given a string, sss , and a list of words, wordswordswords , that are all of the same length. Find all starting indices of substring(s)substring(s)substring(s) in sss that is a concatenation of each word in wordswordswords exactly once and without any intervening characters.

题意

给定一个长为 nnn 的字符串 sss ，给定 mmm 个长度均为 LLL 的字符串 wordswordswords ，求 sss 的所有满足条件的子串的起点 iii ，要求子串由 wordswordswords 中的每个单词不重叠地首尾相接而成。首尾相接的顺序是任意的，每个单词都恰好出现一次，不允许其他无关字符出现在子串中。在 wordswordswords 中可能存在相同的字符串。

题解

因为字符串的长度相同，所以我们完全可以从哈希的角度去考虑，字符串的长度对答案毫无影响。我们不妨先只考虑 L=1L = 1L=1 的情形，那么问题退化为：

  • 给定一个长为 nnn 的数组，指定 mmm 种元素的出现次数，求该数组的所有满足条件的连续子序列，满足其中每个元素的出现次数符合预设指定值。

显然该问题具有单调性，双指针扫描解决即可，时间复杂度为 O(n)O(n)O(n) 。

回到原问题中，我们只需以 LLL 为单位去处理即可。唯一的问题在于，字符串 sss 并不是有和 LLL 相关的规律性的，我们需要枚举求解的起点 left∈[0, L−1]left \in [0,\ L-1]left∈[0, L−1] 而不是简单地令 left=0left = 0left=0 。因此总的时间复杂度为 O(nL)O(nL)O(nL) 。
