The n-queens puzzle is the problem of placing nnn queens on an n×nn \times nn×n chessboard such that no two queens attack each other.

Given an integer nnn , return all distinct solutions to the n-queens puzzle.

Each solution contains a distinct board configuration of the n-queens’ placement, where 'Q' and '.' both indicate a queen and an empty space respectively.

题意

给定一个正整数 nnn ，求 nnn 皇后问题的所有合法解。
