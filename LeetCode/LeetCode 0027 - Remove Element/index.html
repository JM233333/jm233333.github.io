Easy O(n)O(n)O(n) O(1)O(1)O(1)

Given an array numsnumsnums and a value valvalval , remove all instances of that value in-place and return the new length.

Do not allocate extra space for another array, you must do this by modifying the input array in-place with O(1)O(1)O(1) extra memory.

The order of elements can be changed. It doesn’t matter what you leave beyond the new length.
