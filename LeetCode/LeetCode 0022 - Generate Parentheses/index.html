Difficulty Time Complexity Limit Extra-Memory Complexity Limit
Medium O(n∗22n)O(n * 2^{2n})O(n∗22n) or O(n∗C2nn/(n+1))O(n * C_{2n}^{n} / (n + 1))O(n∗C2nn​/(n+1)) −-−

Given nnn pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

For example, given n=3n = 3n=3 , a solution set is:

题意

给定一个正整数 nnn ，求 nnn 对括号的所有合法的排列组合，即所有长为 2n2n2n 的合法的括号匹配字符串。

题解

可以暴力枚举所有可能的 22n2^{2n}22n 种排列，然后判断每种排列是否合法，时间复杂度为 O(n∗22n)O(n * 2^{2n})O(n∗22n) 。

但这显然不是最优解，因为对于本题而言，如果用DFS实现，我们可以进行合法性判断的剪枝，只需要在DFS的过程中统计当前左右括号个数即可。

我们知道 nnn 对括号匹配的合法排列数为第 nnn 项卡特兰数 C2nn/(n+1)C_{2n}^{n} / (n + 1)C2nn​/(n+1) ，因此DFS的时间复杂度为 O(n∗C2nn/(n+1))O(n * C_{2n}^{n} / (n + 1))O(n∗C2nn​/(n+1)) 。
