题面

Medium O(log⁡m+log⁡n)O(\log{m} + \log{n})O(logm+logn) −-−

Write an efficient algorithm that searches for a value in an m×nm \times nm×n matrix. This matrix has the following properties:

  • Integers in each row are sorted from left to right.

  • The first integer of each row is greater than the last integer of the previous row.

题意

给定一个 m×nm \times nm×n 的矩阵，该矩阵的每行从左到右有序递增，且第 iii 行的首元素比第 i−1i - 1i−1 行的末元素大，要求实现在该矩阵中高效地查找某个值是否存在。

题解

由矩阵的有序性质不难想到用二分查找求解。

显然，假设目标值存在，那么在矩阵的第 mmm 列上查找大于等于 targettargettarget 的第一行（事实上与 lower_bound 等价）即可确定对应的行号，再在该行上查找 targettargettarget 是否存在即可，时间复杂度为 O(log⁡m+log⁡n)O(\log{m} + \log{n})O(logm+logn) 。
