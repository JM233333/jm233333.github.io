Given a string s1s1s1 , we may represent it as a binary tree by partitioning it to two non-empty substrings recursively.

Below is one possible representation of $s1 = $ "great" :

To scramble the string, we may choose any non-leaf node and swap its two children.

For example, if we choose the node "gr" and swap its two children, it produces a scrambled string "rgeat" .

We say that "rgeat" is a scrambled string of "great" .

Similarly, if we continue to swap the children of nodes "eat" and "at" , it produces a scrambled string "rgtae" .

We say that "rgtae" is a scrambled string of "great" .

题意

定义字符串 sss 的扰乱如下：

首先把 sss 构造成二叉树的结构，树上的每个节点为 sss 的某个子串，且节点 uuu 所表示的子串为 uuu 的子节点 vlv_lvl​ 和 vrv_rvr​ 所表示的子串左右相连的结果，如此递归构造直至子串长度为 111 ，即作为叶子节点。显然符合条件的树存在多种，如题目样例所示。

可以任意多次交换 sss 的树上的某个非叶子节点的两个子树。最后从左到右读取叶子节点，得到的字符串即为 sss 的扰乱。显然符合条件的扰乱存在多种。

给定两个长度相同的字符串 s1s_1s1​ 和 s2s_2s2​ ，判断 s2s_2s2​ 是否为 s1s_1s1​ 的任意一种可能的扰乱。

题解

考虑动态规划，设 dp[i][j][len]=0/1dp[i][j][len] = 0/1dp[i][j][len]=0/1 表示 s2[j, j+len)s_2[j,\ j + len)s2​[j, j+len) 是否为 s1[i, i+len)s_1[i,\ i + len)s1​[i, i+len) 的某种扰乱。

状态转移方程：

{dp[i][j][1]=(s1[i]==s2[j])dp[i][j][len]=(dp[i][j][k] && dp[i+k][j+k][len−k]) ∣∣    (dp[i][j+len−k][k] && dp[i+k][j][len−k])\begin{cases} dp[i][j][1] &= (s_1[i] == s_2[j])\\ dp[i][j][len] &= (dp[i][j][k]\ \&\&\ dp[i + k][j + k][len - k])\ ||\\ &\ \ \ \ (dp[i][j + len - k][k]\ \&\&\ dp[i + k][j][len - k]) \end{cases}⎩⎪⎨⎪⎧​dp[i][j][1]dp[i][j][len]​=(s1​[i]==s2​[j])=(dp[i][j][k] && dp[i+k][j+k][len−k]) ∣∣    (dp[i][j+len−k][k] && dp[i+k][j][len−k])​

我们考虑枚举当前子串的划分位置 kkk ，那么 s1[i, i+k)s_1[i,\ i + k)s1​[i, i+k) 和 s2[j, j+k)s_2[j,\ j + k)s2​[j, j+k) 相匹配有两种可能的情况，即是否交换左右子树。

  • 如果不交换子树，则 s1[i, i+k)s_1[i,\ i + k)s1​[i, i+k) 与 s2[j, j+k)s_2[j,\ j + k)s2​[j, j+k) 相匹配，而 s1[i+k, i+len)s_1[i + k,\ i + len)s1​[i+k, i+len) 与 s2[j+k, j+len)s_2[j + k,\ j + len)s2​[j+k, j+len) 相匹配；

  • 如果交换子树，则 s1[i, i+k)s_1[i,\ i + k)s1​[i, i+k) 与 s2[j+len−k, j+len)s_2[j + len - k,\ j + len)s2​[j+len−k, j+len) 相匹配，而 s1[i+k, i+len)s_1[i + k,\ i + len)s1​[i+k, i+len) 与 s2[j, j+len−k)s_2[j,\ j + len - k)s2​[j, j+len−k) 相匹配；

由此可得状态转移方程。
