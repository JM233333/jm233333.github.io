题意

给定一个正整数 nnn ，求有多少种不同的由键值为 1,2,...,n1, 2, ..., n1,2,...,n 的 nnn 个节点组成的二叉查找树的结构。

题解

为便于描述，记由键值为 1,2,...,n1, 2, ..., n1,2,...,n 的 nnn 个节点组成的二叉查找树为 TnT_nTn​ ，并记 TnT_nTn​ 的结构种数为 cnc_ncn​ 。

显然，由键值为 a1,a2,...,an (ai−1<ai)a_1, a_2, ..., a_n\ (a_{i - 1} < a_i)a1​,a2​,...,an​ (ai−1​<ai​) 的 nnn 个节点组成的二叉查找树的结构种数也为 cnc_ncn​ ，因为我们可以将 a1,a2,...,ana_1, a_2, ..., a_na1​,a2​,...,an​ 一一映射到 1,2,...,n1, 2, ..., n1,2,...,n 。

由二叉查找树的性质不难发现，若确定 TnT_nTn​ 的根节点为 iii ，则其左子树中的节点被确定为 1,2,...,i−11, 2, ..., i - 11,2,...,i−1 ，右子树中的节点被确定为 i+1,i+2,...,ni + 1, i + 2, ..., ni+1,i+2,...,n 。那么 TnT_nTn​ 以 iii 为根节点的结构种数为其左右子树的结构种数之积，所以 cn=∑i=1nci−1cn−ic_n = \sum_{i = 1}^{n}{c_{i - 1}c_{n - i}}cn​=∑i=1n​ci−1​cn−i​ （特别地，令 c0=1c_0 = 1c0​=1 ）。

考虑动态规划，设 dp[i]=cidp[i] = c_idp[i]=ci​ 。状态转移方程为：
