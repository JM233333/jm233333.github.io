LeetCode 0021 - Merge Two Sorted Lists

Posted on 2019-11-11 In LeetCode

本文为 JM233333 原创所著，遵循 BY-NC-SA 4.0 许可协议，转载请注明出处。如果您认为文中有谬误或对文中内容有疑问，欢迎理性指出，大家共同探讨和进步。

Hints

  • 链表 + 水题 + 双指针扫描

题面

Difficulty Time Complexity Limit Extra-Memory Complexity Limit
Easy O(n)O(n)O(n) −-−

Merge two sorted linked lists and return it as a new list. The new list should be made by splicing together the nodes of the first two lists.

Example:

题意

给定两个有序链表，要求将其中所有节点合并为一个新的有序链表。

题解

水题，双指针扫描即可。

我们同时从两个链表的表头出发，设两个指针为 p, qp,\ qp, q ，根据 p->val 和 q->val 决定将何者放入新链表并递进即可。

细节问题：

  1. 注意特判其中一个或两个链表为空的情况。

  2. 注意可能其中一个指针到达表尾而另一个却没有，要特殊处理剩余部分。
