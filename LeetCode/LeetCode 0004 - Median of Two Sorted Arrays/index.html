LeetCode 0004 - Median of Two Sorted Arrays

题面

Hard O(log⁡(m+n))O(\log{(m + n)})O(log(m+n)) −-−

There are two sorted arrays nums1 and nums2 of size mmm and nnn respectively.

Find the median of the two sorted arrays. The overall run time complexity should be O(log⁡(m+n))O(\log(m + n))O(log(m+n)) .

You may assume nums1 and nums2 cannot be both empty.

题意

给定两个升序排列的数组 nums1 and nums2 ，大小分别为 mmm 和 nnn ，求两个数组中的所有数的中位数。

题解

为便于描述，记 nums1 和 nums2 为 AAA 和 BBB 。

在有序序列中进行查找操作，很容易想到二分查找，然而问题中存在两个序列，虽然问题的单调性是显然的，但是我们却难以直接进行二分。

我们考虑中位数的性质。不妨设 m+nm + nm+n 为奇数，显然，比中位数小的数有 (m+n−1)/2(m + n - 1) / 2(m+n−1)/2 个，算上中位数自己就一共有 (m+n+1)/2(m + n + 1) / 2(m+n+1)/2 个，而这些数有些来自于 AAA 、有些来自于 BBB 。不妨设这些数中有 iii 个来自于 AAA 、有 jjj 个来自于 BBB ，这意味着我们可以将 AAA 和 BBB 各自划分为两个部分：

  • AAA 的左部包括 A0, A1, ..., Ai−1A_0,\ A_1,\ ...,\ A_{i-1}A0​, A1​, ..., Ai−1​ ，右部包括 Ai, Ai+1, ..., Am−1A_i,\ A_{i+1},\ ...,\ A_{m-1}Ai​, Ai+1​, ..., Am−1​ ；

  • BBB 的左部包括 B0, B1, ..., Bj−1B_0,\ B_1,\ ...,\ B_{j-1}B0​, B1​, ..., Bj−1​ ，右部包括 Bj, Bj+1, ..., Bn−1B_j,\ B_{j+1},\ ...,\ B_{n-1}Bj​, Bj+1​, ..., Bn−1​ ；

我们知道 i+j=(m+n+1)/2i + j = (m + n + 1) / 2i+j=(m+n+1)/2 ，那么对于给定的 iii ，我们显然可以得出唯一合法的 jjj 。

当然，我们也可以将中位数归入右部，那么 i+j=(m+n−1)/2i + j = (m + n - 1) / 2i+j=(m+n−1)/2 ，这并不重要。

注意，由于 i∈[0, m−1]i \in [0,\ m-1]i∈[0, m−1] ，当 m>nm > nm>n 时 jjj 可能为负，此时我们不妨交换 AAA 和 BBB ，显然不会影响答案，且可以简化问题的求解过程。

此时我们已经成功地将两个变量 i, ji,\ ji, j 转化为了一个变量 iii ，现在我们就可以像往常那样二分查找了。

因为我们规定比中位数小的数都在左部、比中位数大的数都在右部，因此有 max(Ai−1, Bj−1)≤min(Ai, Bj)max(A_{i-1},\ B_{j-1}) \le min(A_i,\ B_j)max(Ai−1​, Bj−1​)≤min(Ai​, Bj​) 。而 Ai−1≤AiA_{i-1} \le A_iAi−1​≤Ai​ 和 Bj−1≤BjB_{j-1} \le B_jBj−1​≤Bj​ 显然成立，因此只要 Ai−1≤BjA_{i-1} \le B_jAi−1​≤Bj​ 和 Bj−1≤AiB_{j-1} \le A_iBj−1​≤Ai​ 成立，就说明找到了正确的划分，中位数即为 max(Ai−1, Bj−1)max(A_{i-1},\ B_{j-1})max(Ai−1​, Bj−1​) 。

最后我们考虑具体的二分条件，事实上前面的问题都解决后，这部分是最简单的，稍加思考即可。

  • 当 Ai−1>BjA_{i-1} > B_jAi−1​>Bj​ 时，说明 Ai−1A_{i-1}Ai−1​ 的值太大，应使 iii 减小，且 jjj 会随之增大。

  • 当 Bj−1>AiB_{j-1} > A_iBj−1​>Ai​ 时，说明 Ai−1A_{i-1}Ai−1​ 的值太小，应使 iii 增大，且 jjj 会随之减小。

  • 当 Ai−1≤BjA_{i-1} \le B_jAi−1​≤Bj​ 且 Bj−1≤AiB_{j-1} \le A_iBj−1​≤Ai​ 时，求解完成。

不要忘了处理 m+nm + nm+n 为偶数的情形，这很简单，因而不再赘述。注意可以利用除法下取整的特性简化代码实现，不必为奇数和偶数分别写出 jjj 的不同表达式。

总的时间复杂度为 O(log⁡(min(m, n)))O(\log{(min(m,\ n))})O(log(min(m, n))) 。
