LeetCode 0037 - Sudoku Solver

Posted on 2019-11-19 In LeetCode

DFS

Difficulty Time Complexity Limit Extra-Memory Complexity Limit
Hard O(n3∗n!)O(n^3 * n!)O(n3∗n!) or O(n∗n!)O(n * n!)O(n∗n!) −-−

Write a program to solve a Sudoku puzzle by filling the empty cells.

A sudoku solution must satisfy all of the following rules:

  1. Each of the digits 1-9 must occur exactly once in each row.

  2. Each of the digits 1-9 must occur exactly once in each column.

  3. Each of the the digits 1-9 must occur exactly once in each of the 3×33 \times 33×3 sub-boxes of the grid.

Empty cells are indicated by the character '.' .

A sudoku puzzle…

…and its solution numbers marked in red.

Note:

The given board contain only digits 1-9 and the character '.' .

You may assume that the given Sudoku puzzle will have a single unique solution.

The given board size is always 9×99 \times 99×9 .

题意

给定一个 9×99 \times 99×9 的数独，求出该数独的解。输入保证有且仅有一个合法解。

题解

考虑DFS枚举所有可能的情况，然后用合法性剪枝处理掉所有的非法解即可。在存在合法性剪枝的情况下，搜索本身的时间复杂度为 O(n∗n!)O(n * n!)O(n∗n!) 。

注意，一个可能的想法是利用LeetCode 0036中刚写完的数独合法性判断来求解本题，从正确性上来说当然没什么问题，但每次都进行 O(n2)O(n^2)O(n2) 的判定毫无疑问是非常低效的，总的时间复杂度为 O(n3∗n!)O(n^3 * n!)O(n3∗n!) 。不难发现这样会执行很多次重复的合法性判断。

我们应当在DFS的过程中维护每行每列每个方格中的数字出现情况，这样就可以免除大量的重复判断，总的时间复杂度仍为 O(n∗n!)O(n * n!)O(n∗n!) 。。
