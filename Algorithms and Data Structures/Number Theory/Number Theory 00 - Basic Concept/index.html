前言

数论 (Number Theory) 是纯粹数学的分支之一，主要研究整数的性质。在算法竞赛中，初等数论及其算法是重点考察的对象之一。

数论的主要特点是理论多、性质多、难度大、理论性强，要想学好数论，需要先掌握大量的基本概念。

主要参考了 zbh 大佬的课件，与其他的资料和个人的理解合并整理而得。

若 gcd(a, b)=1gcd(a,\ b) = 1gcd(a, b)=1 ，则称 aaa 和 bbb 互质。

同理可定义 nnn 个数的最大公约数和最小公倍数为 gcd(a1,a2,...,an)gcd(a_1, a_2, ..., a_n)gcd(a1​,a2​,...,an​) 和 lcm(a1,a2,...,an)lcm(a_1, a_2, ..., a_n)lcm(a1​,a2​,...,an​) 。

  • 对任意实数 m>0m > 0m>0 ，有 gcd(am, bm)=m∗gcd(a, b)gcd(am,\ bm) = m * gcd(a,\ b)gcd(am, bm)=m∗gcd(a, b) 。

  • 若 c∣a, c∣bc \mid a,\ c \mid bc∣a, c∣b ，则 c∣gcd(a, b)c \mid gcd(a,\ b)c∣gcd(a, b) ；若 a∣c, b∣ca \mid c,\ b \mid ca∣c, b∣c ，则 lcm(a, b)∣clcm(a,\ b) \mid clcm(a, b)∣c 。

  • ∣ab∣=gcd(a, b)∗lcm(a, b)|ab| = gcd(a,\ b) * lcm(a,\ b)∣ab∣=gcd(a, b)∗lcm(a, b)

设 a, b, ma,\ b,\ ma, b, m 是整数， gcd(a,m)=1gcd(a, m) = 1gcd(a,m)=1 ，则 gcd(a, mb)=gcd(a, b)gcd(a,\ mb) = gcd(a,\ b)gcd(a, mb)=gcd(a, b) 。进一步，若 a∣mba \mid mba∣mb ，则 a∣ba \mid ba∣b 。

质数与合数

设 aaa 是正整数， a≥2a \ge 2a≥2 ，若 aaa 除了 111 和 aaa 之外没有别的正因数，则称 aaa 为质数，否则称 aaa 为合数。

设 a, ba,\ ba, b 是整数，若 gcd(a, b)=1gcd(a,\ b) = 1gcd(a, b)=1 ，则称 aaa 和 bbb 互质。

设 aaa 是合数，则 aaa 必有不超过 a\sqrt{a}a​ 的质因数。

在 [1, n][1,\ n][1, n] 范围内的质数个数是 O(n/log⁡n)O(n / \log{n})O(n/logn) 级别的。

设 aaa 是正整数， a≥2a \ge 2a≥2 ，则 aaa 可唯一的表示为 a=∏i=1spikia = \prod_{i = 1}^{s}{p_i^{k_i}}a=∏i=1s​piki​​ ，其中 p1<p2<...<psp_1 < p_2 < ... < p_sp1​<p2​<...<ps​ 均为质数。

设 aaa 是正整数， a≥2a \ge 2a≥2 ，则 aaa 的因数个数为 ∏i=1s(1+ki)\prod_{i = 1}^{s}{(1 + k_i)}∏i=1s​(1+ki​) ， aaa 的所有因数之和为 ∏i=1s[(piki+1−1)/(pi−1)]\prod_{i = 1}^{s}{[(p_i^{k_i + 1} - 1) / (p_i - 1)]}∏i=1s​[(piki​+1​−1)/(pi​−1)] 。

对单个正整数 xxx 进行质因数分解的时间复杂度为 O(x)O(\sqrt{x})O(x​) ，但是如果要预处理区间 [1, n][1,\ n][1, n] 内的数的质因数分解，时间复杂度为 O(nn/log⁡n)O(n\sqrt{n/\log{n}})O(nn/logn​) ，并不能想当然地认为是 O(nn)O(n\sqrt{n})O(nn​) 。

设 aaa 的因数个数为奇数，则 aaa 是完全平方数。

设 aaa 的因数个数为 222 ，则 aaa 是一个质数。

设 aaa 的因数个数为 333 ，则 aaa 是某个质数的平方。

设 aaa 的因数个数为 444 ，则 aaa 是两个不同质数的乘积或某个质数的三次方。这里不能枚举两质数，否则会超时。

设 aaa 的因数个数为 555 ，则 aaa 是某个质数的四次方。
