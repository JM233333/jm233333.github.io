前言

字符串理论 (String Theory) 是计算机科学中极为重要的一部分，处理字符串是一类极为常见的编程问题，因此学会解决字符串问题是很有必要的。而字符串理论除了基础的字符串处理方法，还包括一些高效的算法和数据结构。

相对于数据结构、基础算法、图论等内容而言，字符串理论的入门难度要高得多，这其中最基础的内容，包括KMP算法、AC自动机等，就已经非常难以理解，希望读者做好心理准备。

主要参考了zbh大佬的课件，与个人的理解合并整理而得。

为便于描述，设下文中字符串的下标均为 0..n−10..n-10..n−1 ，其中 nnn 为字符串长度。

基本概念

字符 (character, char) 是一个单独的字符，通常用小写字母 a, b, ca,\ b,\ ca, b, c 或单词缩写 chchch 表示。

字母表 是一个由若干个不同种类的字符组成的集合，又称字符集，通常用 ∑\sum∑ 表示。字母表记录的是在某个研究问题中，所有字符串中可能出现的字符种类数，例如 ∑={a, b, c}\sum = \{a,\ b,\ c\}∑={a, b, c} 表示当前环境下所有的字符串只会由 a, b, ca,\ b,\ ca, b, c 三种字母构成，此时字符串 ab 和 aac 是合法的串，而 adf 是不合法的串。通常用 ∣∑∣|\sum|∣∑∣ 表示字符集 ∑\sum∑ 的大小，即 ∑\sum∑ 所包含的字符种类数。

字符串 (string) 由若干个字符连接而成，通常用大写字母 S, T, A, B, C, X, Y, ZS,\ T,\ A,\ B,\ C,\ X,\ Y,\ ZS, T, A, B, C, X, Y, Z 或用小写字母 sss 或用单词缩写 str, modestr,\ modestr, mode 表示。通常用 ∣S∣|S|∣S∣ 表示字符串 SSS 的长度，即 SSS 所包含的字符个数。当 ∣S∣=1|S| = 1∣S∣=1 时，亦可认为 SSS 是一个字符。和C++等语言的定义不同，为了方便起见，通常我们不在交流中严格区分字符和字符串，而认为字符是长度为 111 的字符串。通常用 Si (0≤i≤∣S∣−1)S_i\ (0 \le i \le |S|-1)Si​ (0≤i≤∣S∣−1) 表示字符串的第 iii 个字符。当然，下标也可以是 1..∣S∣1..|S|1..∣S∣ ，不过本文认为下标是 0..∣S∣−10..|S|-10..∣S∣−1 。

空串 即空的字符串，不包括任何字符，通常用 ε\varepsilonε 表示，且 ∣ε∣=0|\varepsilon| = 0∣ε∣=0 。

设 SSS 和 TTT 是两个字符串，则用 S+TS + TS+T 或 STSTST 表示两串首尾相接形成的新字符串，称为字符串的 连接 。字符串和字符的连接、字符和字符的连接也是类似的，不予赘述。

设 SSS 是一个字符串，则 S[i, j]=Si+Si+1+...+Sj (0≤i≤j≤∣S∣−1)S[i,\ j] = S_i + S_{i+1} + ... + S_j\ (0 \le i \le j \le |S|-1)S[i, j]=Si​+Si+1​+...+Sj​ (0≤i≤j≤∣S∣−1) 是 SSS 的一个 子串 (substring) ，且是一个非空子串。特别地，空串 ε\varepsilonε 是任何字符串的子串，但它通常没有用处，所以我们在多数情况下只考虑非空子串。

设 SSS 是一个字符串，则子串 S[0, i] (0≤i≤∣S∣−1)S[0,\ i]\ (0 \le i \le |S|-1)S[0, i] (0≤i≤∣S∣−1) 是 SSS 的一个 前缀 (prefix) ，记为 prefix[i]prefix[i]prefix[i] 。当 i==∣S∣−1i == |S|-1i==∣S∣−1 时，该前缀事实上就是串 SSS 本身，此时根据实际情况，我们有时认为这也算是一个前缀，有时则认为不是，这只是为了方便解决问题，并不需要纠结。在后缀的定义中有类似的问题，不予赘述。

设 SSS 是一个字符串，则子串 S[i, ∣S∣−1] (0≤i≤∣S∣−1)S[i,\ |S|-1]\ (0 \le i \le |S|-1)S[i, ∣S∣−1] (0≤i≤∣S∣−1) 是 SSS 的一个 后缀 (suffix) ，记为 suffix[i]suffix[i]suffix[i] 。

设 SSS 和 TTT 是字符串，若 TTT 是 SSS 的一个子串，则称 TTT 匹配了 SSS ，即 TTT 在 SSS 中被匹配。称 SSS 为 主串 或文本串，TTT 为 模式串 。判断若干个字符串是否在另一个字符串中被匹配的这一类问题被称为 字符串匹配问题 。若只有一个模式串，则称为单模式匹配问题，否则称为多模式匹配问题。
