前言

本知识板块主要介绍如下内容：

  • 基础的算法思想，包括贪心、动态规划、分治、搜索、二分查找等；

  • 基础算法思想的一些经典运用，包括哈夫曼编码、快速幂、平面最近点对等；

  • 本科课程内没有的基础的算法技巧，包括离散化、二进制压缩、双指针扫描、折半搜索等。

由于动态规划的特殊性（难度大、思维性强、有很多非常巧妙的运用），我将动态规划的一些经典应用放在专门的动态规划的知识板块中介绍，这里的部分只介绍动态规划思想本身（当然，是有实例来辅助阅读和理解的）。至于其他基础算法思想的经典应用都放在本板块内进行介绍。
