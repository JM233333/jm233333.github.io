最短路径树 + DijkstraHeap + 贪心

Time Limit Memory Limit Source
2000 MS 524288 K 2019小学期

题目描述

JM今天过得实在是太惨了，他肥肠不爽，因此他搬出了他祖传的月亮神树。月亮神树的光辉照耀XJTUACM，每分钟都可以从所有被照射的人的手中夺取他1%的财产。JM觉得他很快就能赚得盆满钵满。然而月亮神树不久后突然失控了，不仅不把它夺来的财产交给JM，甚至开始夺取JM的财产，把JM气疯了，赶紧关闭了月亮神树。

因为月亮神树没运行多久就被关掉了，所以大家并没有损失多少，迫于月亮神树的威压，并没有人去理它。可是wzk昨天刚中了彩票，赚了100000000000￥，转眼就被剥夺了很多。因为wzk太rich了，所以他的损失格外大。为了夺回财产，无敌的wzk决定消灭月亮神树。

月亮神树的构造非常神奇，它的枝杈交错纵横，树上甚至存在环路，可以视为一个无向带权连通图的结构。月亮神树有一个核心节点，记为 sss 。要想消灭月亮神树，必须找到月亮神树的严格最不科学生成树，这是它的弱点，这样就能将其一举摧毁。

定义：一个图 GGG 的不科学生成树是 GGG 的一棵子树，在这棵子树上，从核心节点 sss 到任意一个节点 uuu 的最短路径长度，和在原图上是等长的。其中节点 sss 是月亮神树的核心节点。

定义：一个图 GGG 的最不科学生成树是 GGG 的所有不科学生成树中，边权和最小的一棵树。

定义：一个图 GGG 的严格最不科学生成树是 GGG 的所有最不科学生成树中，有序边序列的字典序最小的一棵树。

定义：一个图 GGG 的有序边序列是指，将图上所有边按编号从小到大排序后得到的编号序列。当然，子图子树也适用。

现在给定月亮神树，即给定一个 nnn 个点 mmm 条边的无向带权连通图，点的编号为 1,2,...,n1, 2, ..., n1,2,...,n ，边的编号为 1,2,...,m1, 2, ..., m1,2,...,m ，给定核心节点的编号 sss ，求其严格最不科学生成树。

输入格式

第一行三个正整数 n, m, sn,\ m,\ sn, m, s 。

接下来 mmm 行，第 iii 行三个正整数 u, v, wu,\ v,\ wu, v, w ，表示编号为 iii 的边。

输入保证图是连通的，保证图上不含重边和自环。

输出格式

第一行两个正整数 cntcntcnt 和 sumsumsum ，用空格隔开，其中 cntcntcnt 表示严格最不科学生成树的边的个数， sumsumsum 表示严格最不科学生成树的边权和。

接下来一行 cntcntcnt 个正整数，用空格隔开，为树上所有边的编号，按编号从小到大输出。

题解

本题非原创，原题为CodeForces 545E。

题目显著地要求找到一棵以 sss 为源点的最短路径树，其次找到一棵边权和最小的，再次找到一棵边编号字典序最小的。如果不知道什么是最短路径树请先自行学习。

事实上，后面这些乱七八糟的条件都满足贪心性质，贪心的正确性与Dijkstra的正确性等价。因此只需要写一个Dijkstra的简单变形，然后用它跑出最短路径树即可。
