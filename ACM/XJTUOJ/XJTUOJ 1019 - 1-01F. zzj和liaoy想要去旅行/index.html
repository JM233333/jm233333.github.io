题目描述

zzj和liaoy都很喜欢旅行，他们现在打算计划新的一次旅行。在一番讨论之后他们将景点根据种类的不同，例如海滩、山脉、历史人文景观等等分成了 444 类，同时对于每一个想去的景点他们都有一个权值。我们将这 444 类景点分别记为 A,B,C,DA, B, C, DA,B,C,D 四个数组，数组中记录的是该类的各个景点的权值。

现在zzj和liaoy打算在 444 类景点中各选出一个，使得选出的 444 个景点中，权值最高的在 A,BA, BA,B 两种里面，权值最低的在 C,DC, DC,D 两种里面。也就是说，要选出一个四元组 (a,b,c,d)(a, b, c, d)(a,b,c,d) ，满足 a∈A, b∈B, c∈C, d∈Da \in A,\ b \in B,\ c \in C,\ d \in Da∈A, b∈B, c∈C, d∈D 且 max(a,b,c,d)=a or b, min(a,b,c,d)=c or dmax(a, b, c, d) = a\ or\ b,\ min(a, b, c, d) = c\ or\ dmax(a,b,c,d)=a or b, min(a,b,c,d)=c or d。

现在zzj和liaoy想知道他们一共有多少种符合要求的选法。

输入格式

输入一共四行，每行表示一个种类景点的期望值信息，依次是 A,B,C,DA, B, C, DA,B,C,D 四类。

每行开始是一个整数 nnn ，表示该类景点一共有 nnn 个。接下来包含 nnn 个数字，表示这 nnn 个景点的期望值。

输出格式

输出一行一个非负整数，表示一共有多少种选法。

题解

本题有多种解法，这里只介绍我的折半枚举的做法，时间复杂度为 O(n2log⁡n)O(n^2\log{n})O(n2logn) ，适用于 1≤n≤10001 \le n \le 10001≤n≤1000 且 1≤Ai,Bi,Ci,Di≤10181 \le A_i, B_i, C_i, D_i \le 10^{18}1≤Ai​,Bi​,Ci​,Di​≤1018 的情况。事实上本题有复杂度更优的做法，不过今天学的是折半枚举。

考虑题目条件，答案集事实上等价于下列四种情况的并集：

  1. max=a, min=cmax = a,\ min = cmax=a, min=c

  2. max=a, min=dmax = a,\ min = dmax=a, min=d

  3. max=b, min=cmax = b,\ min = cmax=b, min=c

  4. max=b, min=dmax = b,\ min = dmax=b, min=d

如果直接对每种情况折半求解，显然会有重复。一种解法就是直接求解然后容斥去重。为了避免去重，我们可以将上述四种情况再分解为下面五种情况：

  1. min{a, b}≥max{c, d}min\{a,\ b\} \ge max\{c,\ d\}min{a, b}≥max{c, d}

  2. a≥d>b≥ca \ge d > b \ge ca≥d>b≥c

  3. a≥c>b≥da \ge c > b \ge da≥c>b≥d

  4. b≥d>a≥cb \ge d > a \ge cb≥d>a≥c

  5. b≥c>a≥db \ge c > a \ge db≥c>a≥d

这样就不会有任何重复了。重复的产生主要源于≥\ge≥中的相等的情况，现在这些相等的情况只存在于分类1中。
