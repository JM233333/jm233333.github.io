题目描述

穷的叮当响的jwp去幼稚园做兼职，他决定每天订一些糖果发给小朋友们吃(虽然他已经很穷了，但他还是很想让小朋友开心)，他要做 kkk 天的老师，每天的学生数量是不一定的，但是为了方便，他每天订购的糖果数量是一定的，为了激励小朋友们的积极性，他决定每天表现最不好的 aaa 个小朋友分到的糖果要比其他人少，简单的来说，如果第 iii 天有 AiA_iAi​ 个小朋友，他订购的糖果数量为 XXX ，则 X mod Ai=Ai−aX \bmod A_i = A_i - aXmodAi​=Ai​−a ，(当然也有可能这 aaa 个小朋友分不到任何糖果)，jwp想让你帮他求出他最少需要准备的糖果数量 XXX 。

第一行一个正整数 TTT ，表示数据组数。

第一行两个整数 k, ak,\ ak, a 。

第二行 kkk 个整数，表示每天来的小朋友的数量 AiA_iAi​ 。

输出格式

输出 TTT 行，每一行一个整数，表示最小糖果数量 XXX 。

题解

X mod Ai=Ai−aX \bmod A_i = A_i - aXmodAi​=Ai​−a 等价于 (X+a) mod Ai=0(X + a) \bmod A_i = 0(X+a)modAi​=0 。由于 aaa 是给定的常数，令 Y=X+aY = X + aY=X+a ，问题转化为求最小的 YYY ，使得对于所有的 iii ，有 Ai∣YA_i \mid YAi​∣Y 。那么显然 Y=LCM(A1, A2, ..., Ak)Y = LCM(A_1,\ A_2,\ ...,\ A_k)Y=LCM(A1​, A2​, ..., Ak​) 。
