题目描述

nocriz是一名喜欢旅游的同学，在来到渡渡鸟王国时，坐上了颜色多样的火车。

nocriz同学在车上，车上有 nnn 个车厢，每一个车厢有一种颜色。

他想知道对于每一个正整数 x∈[1, n]x \in [1,\ n]x∈[1, n] ，集合 {(i, x, j) ∣ i<x<j, lx≤coli=colj≤rx}\{ (i,\ x,\ j)\ |\ i < x < j,\ l_x \le col_i = col_j \le r_x \}{(i, x, j) ∣ i<x<j, lx​≤coli​=colj​≤rx​} 中包含多少个元素。

换句话说，就是要求每一个车厢两边有多少对颜色相同的车厢，并且这一对车厢的颜色要在 lxl_xlx​ 到 rxr_xrx​ 之间。其中 colicol_icoli​ 代表 iii 号车厢的颜色， lx, rxl_x,\ r_xlx​, rx​ 代表颜色的限制。

输入格式

第一行一个正整数 nnn 。

第二行 nnn 个三元组，每个三元组包括三个正整数 (coli, li, ri)(col_i,\ l_i,\ r_i)(coli​, li​, ri​) ，输入中没有括号，这 3n3n3n 个正整数之间均只用空格隔开，详见样例。

输出格式

输出一行 nnn 个非负整数代表答案。

题解

这是一道很好的题，思维难度较大。

题目看起来很恶心，但是仔细读读也能想明白题意。可以发现，我们只需要设法维护：对于当前下标 xxx ，每种颜色在 xxx 左右两侧的出现次数的乘积，再对每种颜色的乘积求和即为答案。

具体做法是，考虑从左到右遍历 x∈[1, n]x \in [1,\ n]x∈[1, n] ，其一开两个桶 cntL[c]cnt_L[c]cntL​[c] 和 cntR[c]cnt_R[c]cntR​[c] 维护 xxx 左侧和右侧每种颜色 ccc 的车厢数，其二用树状数组维护前文所述的每种颜色的乘积，也就是令 query(c)query(c)query(c) 的结果为颜色 1,2,...,c1, 2, ..., c1,2,...,c 的每种颜色的乘积之和。这样一来，对于当前下标 xxx ，显然树状数组的区间查询结果 query(rx)−query(lx−1)query(r_x) - query(l_x - 1)query(rx​)−query(lx​−1) 就是下标 xxx 的答案。

问题在于如何随着 xxx 的变化来维护两个桶和树状数组中记录的乘积值。当 xxx 从 x=i−1x = i - 1x=i−1 变化到 x=ix = ix=i 时，容易发现，需要经过三个阶段完成答案的计算：

  1. 将车厢 iii 从右侧移除，那么 cntR[coli]cnt_R[col_i]cntR​[coli​] 的值会减去 111 ，且颜色 colicol_icoli​ 的成绩会减去 cntL[coli]cnt_L[col_i]cntL​[coli​] 。

  2. 求解 x=ix = ix=i 的答案。

  3. 将车厢 iii 移入左侧，那么 cntL[coli]cnt_L[col_i]cntL​[coli​] 的值会加上 111 ，且颜色 colicol_icoli​ 的成绩会加上 cntR[coli]cnt_R[col_i]cntR​[coli​] 。

这样一来问题就解决了，时间复杂度为 O(nlog⁡n)O(n\log{n})O(nlogn) 。
