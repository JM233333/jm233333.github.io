题目描述

在学习了今天的课程以后，tyx也对N皇后问题产生了兴趣，可惜他想了很久还是不知道怎么写，于是又一次抱住你的大腿请求你的帮助。

他想知道对于 N×NN \times NN×N 的棋盘，最多有多少种满足条件的摆法？

所谓满足条件的摆法，即每一行、每一列、每一条对角线最多仅能有一个皇后。

当然，tyx为了确保你不是随便报了一个数字来忽悠他，他需要你给出按照字典序从小到大排序的前 333 组解（如果不足 333 种，则全部输出）。

一个解的序列是这样的定义的：设 aia_iai​ 表示第 iii 行的皇后放在了第 aia_iai​ 列的位置，其中 1≤i≤N, 1≤ai≤N1 \le i \le N,\ 1 \le a_i \le N1≤i≤N, 1≤ai​≤N ，那么这个解所对应的的序列就是 a1, a2, ..., ana_1,\ a_2,\ ...,\ a_na1​, a2​, ..., an​ 。

注意，tyx非常讨厌数学好的人，因此你如果直接用公式计算出了答案，tyx可能会打死你。

输入格式

一行一个正整数 NNN 。

输出格式

首先是最多 333 行，表示字典序最小的前 333 组解。若解的总数不足 333 ，则输出所有的解即可。

接下来一行一个整数 xxx ，表示满足条件的摆法总数。请不要用公式计算答案哦~
