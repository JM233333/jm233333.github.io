题目描述

不知道已经成为大学生的你是否还记得中位数这个东西，我们似乎很少再用到它，今天JM就想让你做一下这个小学数学题。

初始你手里什么也没有，接下来JM会按顺序给你 nnn 个数。当你手中的数的个数为奇数时，你需要告诉JM你手里这堆数的中位数是多少。

输入格式

第一行一个正整数 nnn ，表示给你的数的个数。

接下来一行 nnn 个整数，表示依次给你的这些数 aia_iai​ 。

输出格式

输出 xxx 行，每行一个整数，表示答案。

当 nnn 为奇数时 x=n/2+1x = n/2 + 1x=n/2+1 ，当 nnn 为偶数时 x=n/2x = n/2x=n/2 ，其中除法为下取整。

题解

本题是堆（优先队列）的一类经典应用，用两个堆来维护答案。

我们用一个大顶堆 pqLpq_LpqL​ 来维护较小的那一半的数，用一个小顶堆 pqRpq_RpqR​ 来维护较大的那一半的数，那么对于每个被插入的数 aia_iai​ ，我们就可以 O(log⁡n)O(\log{n})O(logn) 地维护这两个堆。

至于具体实现，当 aia_iai​ 到来时，如果 ai>pqR.top()a_i > pq_R.top()ai​>pqR​.top() ，意味着它应该被压入较大的一边，否则它应该被压入较小的一边。在将 aia_iai​ 入队后再动态调整两个堆的大小，保证一边一半即可，这很容易，只需要将多的那一堆的堆顶取出放入另一堆中即可。

当 iii 为奇数时，多的那个数放在左右都无所谓，取决于你的实现，那么答案即为 pqLpq_LpqL​ 的堆顶或是 pqRpq_RpqR​ 的堆顶。详见代码实现。
