题目描述

jwp竟然兼职做导游！某天，jwp带着 nnn 个游客去动物园游玩，但是可恶的园长不予许jwp统一买票，所以所有的游客只能排队买票。

每一个游客的脾气和办事效率是不一样的，意味着每一个游客有一个暴躁值 aia_iai​ 和一个买票需要花费的时间 bib_ibi​（即他从开始买票到买票结束所花费的时间） ，这个游客最终的不愉快值可以表示为他的暴躁值 aia_iai​ 乘以排队及买票花费的总时间（即从第一个游客开始买票，到第 iii 个游客买票结束）。假设每一个游客买完票会立刻离开，同时下一个游客开始买票。jwp决定安排游客的买票顺序，从而使所有游客的不愉快值之和最小化。他想知道这个最小值是多少。

输入格式

第一行一个正整数 nnn 。

接下来 nnn 行，每行两个正整数 ai, bia_i,\ b_iai​, bi​ 。

输出格式

一行一个整数表示答案。

题解

考虑贪心，使用相邻交换法证明其正确性。为便于描述，设游客的编号为 1,2,...,n1, 2, ..., n1,2,...,n 。

根据题意，第 iii 名游客的不愉快值为 ai∗∑k=1ibka_i * \sum_{k = 1}^{i}{b_k}ai​∗∑k=1i​bk​ 。

考虑两名相邻的游客 iii 和 jjj (i, j∈[1, n], ∣i−j∣=1)(i,\ j \in [1,\ n],\ |i - j| = 1)(i, j∈[1, n], ∣i−j∣=1) ，当 i<ji < ji<j 时，两名游客的总不愉快值为 ai∗∑k=1ibk+aj∗∑k=1jbka_i * \sum_{k = 1}^{i}{b_k} + a_j * \sum_{k = 1}^{j}{b_k}ai​∗∑k=1i​bk​+aj​∗∑k=1j​bk​ ，即 ai∗(bi+∑k=1i−1bk)+aj∗(bi+bj+∑k=1i−1bk)a_i * (b_i + \sum_{k = 1}^{i-1}{b_k}) + a_j * (b_i + b_j + \sum_{k = 1}^{i-1}{b_k})ai​∗(bi​+∑k=1i−1​bk​)+aj​∗(bi​+bj​+∑k=1i−1​bk​) 。当 j<ij < ij<i 时类同，不予赘述。为便于描述，记 sum=∑k=1jbksum = \sum_{k = 1}^{j}{b_k}sum=∑k=1j​bk​ 。

设 i<ji < ji<j 的情况优于 j<ij < ij<i 的情况，则：

ai∗(bi+sum)+aj∗(bi+bj+sum)<aj∗(bj+sum)+ai∗(bj+bi+sum)a_i * (b_i + sum) + a_j * (b_i + b_j + sum) < a_j * (b_j + sum) + a_i * (b_j + b_i + sum)ai​∗(bi​+sum)+aj​∗(bi​+bj​+sum)<aj​∗(bj​+sum)+ai​∗(bj​+bi​+sum)

⇒aibi+ajbi+ajbj<ajbj+aibj+aibi\Rightarrow a_ib_i + a_jb_i + a_jb_j < a_jb_j + a_ib_j + a_ib_i⇒ai​bi​+aj​bi​+aj​bj​<aj​bj​+ai​bj​+ai​bi​

⇒ajbi<aibj\Rightarrow a_jb_i < a_ib_j⇒aj​bi​<ai​bj​

因此以该式为依据对游客进行排序即可得到最优解。

注意答案累加的结果可能会爆int。
