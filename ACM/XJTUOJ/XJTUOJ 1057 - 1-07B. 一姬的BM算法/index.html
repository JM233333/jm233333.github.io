题目描述

一姬又学习了用于 O(n2)O(n^2)O(n2) 求解递推数列最短递推式的算法：BerlekampMassy算法（简称BM算法），具体请见：毛啸《关于数列递归式的一些研究》。

好吧，以上是一姬骗你的，事实上BM算法是一种朴素的字符串匹配算法，用于求解以下问题：

给定一个字符串 AAA ，和另一个字符串 BBB ，问 BBB 是否是 AAA 的某个子串？

cyy觉得这题太简单了，于是把它改了一下：

给定一个环形字符串 AAA ，和一个字符串 BBB ，问 BBB 是否是 AAA 的某个子串？

环形字符串表示将一个字符串首尾连接形成的环状结构，例如cde并没有在链式字符串deaaaaaac中出现，但是它可以在环形字符串deaaaaaac中出现，也可以在环形字符串caaaaaaed中出现。

请你帮一姬解决这个问题。

输入格式

第一行一个只有小写英文字母的字符串 AAA 。

第二行一个只有小写英文字母的字符串 BBB 。

输出格式

如果 BBB 是 AAA 的某个子串，输出“Yes”，否则输出“No”（不包括引号，注意字母大小写）。

题解

显然可以用KMP算法求解，只要处理好环形字符串即可。

事实上，我们可以使用通用的处理环形序列的方法。对于一个长为 nnn 的成环的序列 aaa ，我们只需将其变换为长为 2n−12n - 12n−1 的序列 {a1, a2, ..., an−1, an, a1, a2, ..., an−1}\{a_1,\ a_2,\ ...,\ a_{n-1},\ a_n,\ a_1,\ a_2,\ ...,\ a_{n-1}\}{a1​, a2​, ..., an−1​, an​, a1​, a2​, ..., an−1​} ，即可将问题等价转换为链状序列上的问题，直接在该序列上跑KMP即可。

注意，根据题意反过来匹配成功也是可以的，所以要把变换后的 AAA 反过来再跑一遍KMP。
