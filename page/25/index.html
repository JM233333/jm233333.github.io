LeetCode 0046 - Permutations

题面

Difficulty Time Complexity Limit Extra-Memory Complexity Limit
Medium O(n!)O(n!)O(n!) −-−

Given a collection of numbers that might contain duplicates, return all possible unique permutations.

Example:

题意

给定一个长为 nnn 的数组（可能存在重复元素），求其所有本质不同的排列（即对结果进行去重）。

题解

用 STL 的 next_permutation 直接求解即可。该函数的内部实现包括了去重处理，不需要我们再进行额外的去重操作。代码和 LeetCode 0046 相同，此处就不再重复贴出了。

如果禁止使用这一标准库函数，我们可以 DFS 遍历全排列然后直接用 set 暴力去重，但是这样做的时间复杂度会很差。我们有必要先对数组排序，然后在 DFS 的过程中去重。

题意

给定 nnn 个字符串，称一个字符串的本质串为将其内部字母排序后得到的字符串，要求将所有本质相同的字符串放在一个 vector 内，返回所有的 vector 。
