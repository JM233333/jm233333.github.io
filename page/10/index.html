KMP

Time Limit Memory Limit Source
1000 MS 524288 K 2019小学期

题目描述

一姬又学习了用于 O(n2)O(n^2)O(n2) 求解递推数列最短递推式的算法：BerlekampMassy算法（简称BM算法），具体请见：毛啸《关于数列递归式的一些研究》。

好吧，以上是一姬骗你的，事实上BM算法是一种朴素的字符串匹配算法，用于求解以下问题：

给定一个字符串 AAA ，和另一个字符串 BBB ，问 BBB 是否是 AAA 的某个子串？

cyy觉得这题太简单了，于是把它改了一下：

给定一个环形字符串 AAA ，和一个字符串 BBB ，问 BBB 是否是 AAA 的某个子串？

环形字符串表示将一个字符串首尾连接形成的环状结构，例如cde并没有在链式字符串deaaaaaac中出现，但是它可以在环形字符串deaaaaaac中出现，也可以在环形字符串caaaaaaed中出现。

请你帮一姬解决这个问题。

输入格式

第一行一个只有小写英文字母的字符串 AAA 。

第二行一个只有小写英文字母的字符串 BBB 。

输出格式

如果 BBB 是 AAA 的某个子串，输出“Yes”，否则输出“No”（不包括引号，注意字母大小写）。

样例

样例输入

样例输出

数据范围与提示

1≤∣B∣<∣A∣≤1061 \le |B| < |A| \le 10^61≤∣B∣<∣A∣≤106

题解

显然可以用KMP算法求解，只要处理好环形字符串即可。

事实上，我们可以使用通用的处理环形序列的方法。对于一个长为 nnn 的成环的序列 aaa ，我们只需将其变换为长为 2n−12n - 12n−1 的序列 {a1, a2, ..., an−1, an, a1, a2, ..., an−1}\{a_1,\ a_2,\ ...,\ a_{n-1},\ a_n,\ a_1,\ a_2,\ ...,\ a_{n-1}\}{a1​, a2​, ..., an−1​, an​, a1​, a2​, ..., an−1​} ，即可将问题等价转换为链状序列上的问题，直接在该序列上跑KMP即可。

注意，根据题意反过来匹配成功也是可以的，所以要把变换后的 AAA 反过来再跑一遍KMP。

Time Limit Memory Limit Source
3000 MS 524288 K 2019小学期

题目描述

一姬想要设计一种机器，使自己不断地和牌上分，但是她觉得麻将是可以让四个人都得到快乐的，至少她不希望雀魂玩家减一这种事情的发生。因此她不希望有人被飞，所以她想要你设计一款三倍满自动机，使得自己在闲家和牌收益最大又不会让人被飞。

由于她只和三倍满，所以做出决策以及提高打点是十分重要的，在自动机中，这需要解决以下问题实现：

给定 nnn 个非负整数 a1, a2, ..., ana_1,\ a_2,\ ...,\ a_na1​, a2​, ..., an​ ，给定一个非负整数 xxx ，求 ai⊕aj⊕x (i≠j)a_i \oplus a_j \oplus x\ (i \ne j)ai​⊕aj​⊕x (i​=j) 的最大值，其中 ⊕\oplus⊕ 表示异或运算。

输入格式

第一行一个正整数 nnn 和一个非负整数 xxx 。

第二行 nnn 个非负整数表示 aia_iai​ 。

输出格式

输出一行一个非负整数表示答案。

样例

样例输入

样例输出

数据范围与提示

2≤n≤1062 \le n \le 10^62≤n≤106

0≤x≤1090 \le x \le 10^90≤x≤109

0≤ai≤1090 \le a_i \le 10^90≤ai​≤109

题解

处理二进制数是Trie树的常见应用，通常我们称之为01-Trie。

我们用Trie树储存每个二进制数，并且是按照从高位到低位的顺序从根节点向深层节点存储。因为要求异或值最大，肯定是尽可能让高位的异或结果更大。

求三个数异或的最大值并不好做。我们将问题转化一下，在插入 aia_iai​ 前，在Trie树中查找 ai⊕xa_i \oplus xai​⊕x 与树中已有的某个二进制数 aj (j∈[1, i−1])a_j\ (j \in [1,\ i-1])aj​ (j∈[1, i−1]) 的最大值。

至于在查找时使亦或值最大就简单了。设被查询值为 y=ai⊕xy = a_i \oplus xy=ai​⊕x ，只要尽可能沿着与 yyy 的当前位不同的方向走即可。

时间复杂度为 O(nlog⁡ai)O(n\log{a_i})O(nlogai​) 。
