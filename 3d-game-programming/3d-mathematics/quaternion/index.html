    3D Mathematics 04 - Quaternion

    # 概述

    四元数 (Quaternion) 是3D引擎中另一种用于表示旋转的数学实体，在许多情况下，四元数要优于旋转矩阵，因为它使用更少的存储空间，在进行连接时需要更少的计算，并且更易于进行插值以产生平滑动画。

    四元数的集合由 H\Bbb{H}H 表示，它可以被认为是一个四维向量空间，数学家称之为 汉密尔顿四元数环 (the ring of Hamiltonian quaternions) 。

    起初，四元数并不是专用于旋转的。作为用于描述现实空间的坐标表示方式，数学家Hamilton在复数的基础上创造了四元数，并以四元数的形式来表示空间中一点的坐标，那时向量和矩阵还未被推广使用。后来，四元数的地位已经几乎被向量和矩阵所取代。

    但是在表示旋转时，相对于旋转矩阵和欧拉角而言，四元数具有独特的优势，因而现在四元数主要还是被用于表示旋转。

    # 定义

    我们定义一个四元数为：

    q=⟨w,x,y,z⟩=w+xi+yj+zkq = \langle w,\ x,\ y,\ z \rangle = w + xi + yj + zkq=⟨w, x, y, z⟩=w+xi+yj+zk

    其中 i,j,ki,\ j,\ ki, j, k 作为一种特殊的虚数单位参与运算，并遵循如下规则：

    • i0=j0=k0=1i^0 = j^0 = k^0 = 1i0=j0=k0=1

    • i2=j2=k2=−1i^2 = j^2 = k^2 = -1i2=j2=k2=−1

    • ij=−ji=kij = -ji = kij=−ji=k

    • jk=−kj=ijk = -kj = ijk=−kj=i

    • ki=−ik=jki = -ik = jki=−ik=j

    也可以用标量-向量表示法来描述，这种表示形式和复数非常相似：

    q=s+Vq = s + Vq=s+V

    其中 sss 表示 qqq 的标量部分，即 www ，称为四元数的 实部 (real component) ，而 VVV 表示 qqq 的向量部分，即 x,y,zx,\ y,\ zx, y, z ，称为四元数的 虚部 (imaginary component) 。

    类似于复数，我们定义四元数 q=s+Vq = s + Vq=s+V 的 共轭 (conjugate) 为：

    qˉ=s−V\bar{q} = s - Vqˉ​=s−V

    当四元数的实部 s=0s = 0s=0 时，称 q=xi+yj+zkq = xi + yj + zkq=xi+yj+zk 为纯四元数，这有些类似于纯虚数的定义。它可以用于表示三维空间中的一点 (x,y,z)(x,\ y,\ z)(x, y, z) 。

    # 四元数的基本运算

    设 qqq 是一个四元数，aaa 是一个标量 (Scalar) ，定义四元数的标量乘法为：

    aq=qa=as+aVaq = qa = as + aVaq=qa=as+aV

    q1+q2=(s1+s2)+(V1+V2)q_1 + q_2 = (s_1 + s_2) + (V_1 + V_2)q1​+q2​=(s1​+s2​)+(V1​+V2​)

    q1−q2=q1+(−q2)q_1 - q_2 = q_1 + (-q_2)q1​−q2​=q1​+(−q2​)

    定义四元数的点积为：

    q1⋅q2=s1s2+V1⋅V2q_1 \cdot q_2 = s_1s_2 + V_1 \cdot V_2q1​⋅q2​=s1​s2​+V1​⋅V2​

    q1×q2=V1×V2q_1 \times q_2 = V_1 \times V_2q1​×q2​=V1​×V2​

    ∥q∥=s2+x2+y2+z2\|q\| = \sqrt{s^2 + x^2 + y^2 + z^2}∥q∥=s2+x2+y2+z2​

    这些运算都和向量的运算基本一致，因而不再赘述。并且随着四元数不再被用于表示坐标，这些运算在3D引擎中事实上几乎用不到，只做了解即可。四元数最重要也是最复杂的运算是乘法运算。

    # 四元数的乘法

    虽然和复数很相似，但是四元数的乘法不满足交换律。

    设 q1=w1+x1i+y1j+z1k,q2=w2+x2i+y2j+z2kq_1 = w_1 + x_1i + y_1j + z_1k,\ q_2 = w_2 + x_2i + y_2j + z_2kq1​=w1​+x1​i+y1​j+z1​k, q2​=w2​+x2​i+y2​j+z2​k ，定义四元数的乘法如下：

    也可以用标量-向量表示法来描述，设 q1=s1+V1,q2=s2+V2q_1 = s_1 + V_1,\ q_2 = s_2 + V_2q1​=s1​+V1​, q2​=s2​+V2​ ，则：

    q1q2=(s1s2−V1⋅V2)+(s1V2+s2V1+V1×V2)q_1q_2 = (s_1s_2 - V_1 \cdot V_2) + (s_1V_2 + s_2V_1 + V_1 \times V_2)q1​q2​=(s1​s2​−V1​⋅V2​)+(s1​V2​+s2​V1​+V1​×V2​)

    该式可以由上式根据点积和叉积的运算律替换得到。

    四元数 qqq 与其共轭 qˉ\bar{q}qˉ​ 的乘积等于 qqq 和自身的点积，也就等于其模的平方 ∥q∥2\|q\|^2∥q∥2 。也就是说：

    qqˉ=qˉq=q⋅q=∥q∥2q\bar{q} = \bar{q}q = q \cdot q = \|q\|^2qqˉ​=qˉ​q=q⋅q=∥q∥2

    这可以帮助我们定义四元数的逆：

    q−1=qˉ∥q∥2q^{-1} = \frac{\bar{q}}{\|q\|^2}q−1=∥q∥2qˉ​​

    证明很简单：

    qq−1=qqˉ∥q∥2=∥q∥2∥q∥2=1qq^{-1} = \frac{q\bar{q}}{\|q\|^2} = \frac{\|q\|^2}{\|q\|^2} = 1qq−1=∥q∥2qqˉ​​=∥q∥2∥q∥2​=1
    q−1q=qˉq∥q∥2=∥q∥2∥q∥2=1q^{-1}q = \frac{\bar{q}q}{\|q\|^2} = \frac{\|q\|^2}{\|q\|^2} = 1q−1q=∥q∥2qˉ​q​=∥q∥2∥q∥2​=1

    另外，四元数的乘积的模满足如下规律：

    ∥q1q2∥=∥q1∥∥q2∥\|q_1q_2\| = \|q_1\|\|q_2\|∥q1​q2​∥=∥q1​∥∥q2​∥

    # 四元数和旋转

    # 用四元数表示旋转

    给定一个任意的向量 PPP ，我们要将其绕单位向量 AAA 旋转 θ\thetaθ 角度，这样的旋转可以由一个旋转矩阵来表示。现在我们考虑如何用四元数来表示。

    用于表示三维空间中的旋转的四元数的一般形式为：

    q=cos⁡θ2+Asin⁡θ2=cos⁡θ2+(Axi+Ayj+Azk)sin⁡θ2\begin{aligned} q &= \cos\frac{\theta}{2} + A\sin\frac{\theta}{2} \\ &= \cos\frac{\theta}{2} + (A_xi + A_yj + A_zk)\sin\frac{\theta}{2} \end{aligned}q​=cos2θ​+Asin2θ​=cos2θ​+(Ax​i+Ay​j+Az​k)sin2θ​​

    因为 AAA 是一个单位向量，所以 qqq 是一个单位四元数。定义对 PPP 应用四元数 qqq 所表示的旋转为 Rq(P)R_q(P)Rq​(P) ，则：

    P′=Rq(P)=qPq−1P' = R_q(P) = qPq^{-1}P′=Rq​(P)=qPq−1

    注意，为了与 qqq 进行乘法运算，我们将 PPP 视为一个纯四元数。前文已经讲过，纯四元数亦可用来表达空间中一点的坐标，与向量的表示是等价的。

    下面我们证明 qPq−1qPq^{-1}qPq−1 仍然是一个纯四元数。同样是为了方便参与运算，这里我们将 V=Asin⁡θ2V = A\sin\frac{\theta}{2}V=Asin2θ​ 也视为一个纯四元数。

    qPq−1=(s+V)P(s−V)=(−V⋅P+sP+V×P)(s−V)=−s(V⋅P)+s2P+s(V×P)+(V⋅P)V−sPV−(V×P)V=−s(V⋅P)+s2P+s(V×P)+(V⋅P)V−s(−P⋅V+P×V)−(−(V×P)⋅V+(V×P)×V)=s2P+2s(V×P)+(V⋅P)V−(V×P)×V\begin{aligned} qPq^{-1} &= (s + V)P(s - V) \\ &= (-V \cdot P + sP + V \times P)(s - V) \\ &= -s(V \cdot P) + s^2P + s(V \times P) + (V \cdot P)V - sPV - (V \times P)V \\ &= -s(V \cdot P) + s^2P + s(V \times P) + (V \cdot P)V - s(-P \cdot V + P \times V) - (-(V \times P) \cdot V + (V \times P) \times V) \\ &= s^2P + 2s(V \times P) + (V \cdot P)V - (V \times P) \times V \end{aligned}qPq−1​=(s+V)P(s−V)=(−V⋅P+sP+V×P)(s−V)=−s(V⋅P)+s2P+s(V×P)+(V⋅P)V−sPV−(V×P)V=−s(V⋅P)+s2P+s(V×P)+(V⋅P)V−s(−P⋅V+P×V)−(−(V×P)⋅V+(V×P)×V)=s2P+2s(V×P)+(V⋅P)V−(V×P)×V​

    根据向量的叉积的运算律 (P×Q)×P=∥P∥2Q−(P⋅Q)P(P \times Q) \times P = \|P\|^2Q - (P \cdot Q)P(P×Q)×P=∥P∥2Q−(P⋅Q)P 可得：

    qPq−1=s2P+2s(V×P)+(V⋅P)V−∥V∥2P+(V⋅P)V=(s2−∥V∥2)P+2s(V×P)+2(V⋅P)V\begin{aligned} qPq^{-1} &= s^2P + 2s(V \times P) + (V \cdot P)V - \|V\|^2P + (V \cdot P)V \\ &= (s^2 - \|V\|^2)P + 2s(V \times P) + 2(V \cdot P)V \end{aligned}qPq−1​=s2P+2s(V×P)+(V⋅P)V−∥V∥2P+(V⋅P)V=(s2−∥V∥2)P+2s(V×P)+2(V⋅P)V​

    设 V=tAV = tAV=tA ，则：

    qPq−1=(s2−t2)P+2st(A×P)+2t2(A⋅P)AqPq^{-1} = (s^2 - t^2)P + 2st(A \times P) + 2t^2(A \cdot P)AqPq−1=(s2−t2)P+2st(A×P)+2t2(A⋅P)A

    根据我们在推导旋转矩阵时得到的结果：

    P′=Pcos⁡θ+(A×P)sin⁡θ+A(P⋅A)(1−cos⁡θ)P' = P\cos\theta + (A \times P)\sin\theta + A(P \cdot A)(1 - \cos\theta)P′=Pcosθ+(A×P)sinθ+A(P⋅A)(1−cosθ)

    s2−t2=cos⁡θ2st=sin⁡θ2t2=1−cos⁡θ\begin{aligned} s^2 - t^2 &= \cos\theta \\ 2st &= \sin\theta \\ 2t^2 &= 1 - \cos\theta \end{aligned}s2−t22st2t2​=cosθ=sinθ=1−cosθ​

    s=cos⁡θ2,t=sin⁡θ2s = \cos\frac{\theta}{2},\ t = \sin\frac{\theta}{2}s=cos2θ​, t=sin2θ​

    # 正确性证明

    三维空间中的旋转可以被看作是一个函数 φ\varphiφ ，该函数是一个从 R3\Reals^3R3 到 R3\Reals^3R3 的映射，且其变换结果必须保持长度不变、保持角度不变、保持手性不变。

    对于三维空间中的一点 PPP ，旋转保持长度 ∥P∥\|P\|∥P∥ 当且仅当：

    ∥φ(P)∥=∥P∥\|\varphi(P)\| = \|P\|∥φ(P)∥=∥P∥

    对于三维空间中的两点 P1,P2P_1,\ P_2P1​, P2​ ，旋转保持角度 ∠(OP1,OP2)\angle(OP_1,\ OP_2)∠(OP1​, OP2​) 当且仅当：

    φ(P1)⋅φ(P2)=P1⋅P2\varphi(P_1) \cdot \varphi(P_2) = P_1 \cdot P_2φ(P1​)⋅φ(P2​)=P1​⋅P2​

    对于三维空间中的两点 P1,P2P_1,\ P_2P1​, P2​ ，旋转保持手性当且仅当：

    φ(P1)×φ(P2)=φ(P1×P2)\varphi(P_1) \times \varphi(P_2) = \varphi(P_1 \times P_2)φ(P1​)×φ(P2​)=φ(P1​×P2​)

    我们将函数 φ\varphiφ 扩展为一个从 H\Bbb{H}H 到 H\Bbb{H}H 的映射，定义 φ(s+V)=s+φ(V)\varphi(s + V) = s + \varphi(V)φ(s+V)=s+φ(V) ，那么我们可以将旋转保持角度的条件改写为：

    φ(P1)⋅φ(P2)=φ(P1⋅P2)\varphi(P_1) \cdot \varphi(P_2) = \varphi(P_1 \cdot P_2)φ(P1​)⋅φ(P2​)=φ(P1​⋅P2​)

    由于现在值域为 H\Bbb{H}H ，自变量 P1,P2P_1,\ P_2P1​, P2​ 可以被视为纯四元数。

    根据四元数的乘法运算律，有 P1P2=−V1⋅V2+V1×V2P_1P_2 = - V_1 \cdot V_2 + V_1 \times V_2P1​P2​=−V1​⋅V2​+V1​×V2​ ，由此可以将旋转保持角度的条件和旋转保持手性的条件合并为一个表达式：

    φ(P1)φ(P2)=φ(P1P2)\varphi(P_1)\varphi(P_2) = \varphi(P_1P_2)φ(P1​)φ(P2​)=φ(P1​P2​)

    现在我们证明 Rq(P)R_q(P)Rq​(P) 满足条件。

    对于保持长度：

    ∥Rq(P)∥=∥qPq−1∥=∥q∥∥P∥∥q−1∥=∥P∥∥q∥∥qˉ∥∥q∥2=∥P∥\|R_q(P)\| = \|qPq^{-1}\| = \|q\|\|P\|\|q^{-1}\| = \|P\|\frac{\|q\|\|\bar{q}\|}{\|q\|^2} = \|P\|∥Rq​(P)∥=∥qPq−1∥=∥q∥∥P∥∥q−1∥=∥P∥∥q∥2∥q∥∥qˉ​∥​=∥P∥

    对于保持角度和手性：

    Rq(P1)Rq(P2)=qP1q−1qP2q−1=qP1P2q−1=Rq(P1P2)R_q(P_1)R_q(P_2) = qP_1q^{-1}qP_2q^{-1} = qP_1P_2q^{-1} = R_q(P_1P_2)Rq​(P1​)Rq​(P2​)=qP1​q−1qP2​q−1=qP1​P2​q−1=Rq​(P1​P2​)

    得证。

    # 四元数与旋转矩阵

    # 线性插值

    因为四元数可以看作是由四维向量表示的，所以它们非常适合进行插值。当我们对一个对象进行动画处理时，插值对于生成落在预先计算的关键帧之间的中间状态很有用。
