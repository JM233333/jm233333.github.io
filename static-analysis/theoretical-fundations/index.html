# 偏序集和偏序关系

# 基本概念

定义一个 偏序集 (partial-order set, poset) 为一个二元组 (P,⊑)(P,\ \sqsubseteq)(P, ⊑) ，其中 PPP 是一个集合，⊑\sqsubseteq⊑ 是定义在 PPP 上的偏序关系。

偏序关系 (partial ordering) 是满足如下性质的二元关系：

  • 自反性 (reflexivity) ：∀x∈P,x⊑x\forall x \in P,\ x \sqsubseteq x∀x∈P, x⊑x

  • 反对称性 (anti-symmetry) ：∀x,y∈P,(x⊑y∧y⊑x)⇒(x=y)\forall x,\ y \in P,\ (x \sqsubseteq y \land y \sqsubseteq x) \rArr (x = y)∀x, y∈P, (x⊑y∧y⊑x)⇒(x=y)

  • 传递性 (transitivity) ：∀x,y,z∈P,(x⊑y∧y⊑z)⇒(x⊑z)\forall x,\ y,\ z \in P,\ (x \sqsubseteq y \land y \sqsubseteq z) \rArr (x \sqsubseteq z)∀x, y, z∈P, (x⊑y∧y⊑z)⇒(x⊑z)

几个有助于理解偏序关系的示例：

  • 设 ZZZ 是若干个整数构成的集合，则 (Z,≤)(Z,\ \leq)(Z, ≤) 是一个偏序集，而 (Z,<)(Z,\ <)(Z, <) 不是。

  • 设 SSS 是若干个字符串构成的集合，则 (S,substring)(S,\ substring)(S, substring) 是一个偏序集。

  • 设 P(S)={Si∣Si⊆S}P(S) = \{S_i \mid S_i \subseteq S\}P(S)={Si​∣Si​⊆S} ，则 (P(S),⊆)(P(S),\ \subseteq)(P(S), ⊆) 是一个偏序集。

# 上下界

设 (P,⊑)(P,\ \sqsubseteq)(P, ⊑) 是一个偏序集，对于一个子集 S⊆PS \subseteq PS⊆P ，我们有如下定义：

  • 设 u∈Pu \in Pu∈P ，若 ∀x∈S,x⊑u\forall x \in S,\ x \sqsubseteq u∀x∈S, x⊑u ，则称 uuu 是 SSS 的一个 上界 (upper bound) 。

  • 设 l∈Pl \in Pl∈P ，若 ∀x∈S,l⊑x\forall x \in S,\ l \sqsubseteq x∀x∈S, l⊑x ，则称 lll 是 SSS 的一个 下界 (lower bound) 。

  • 定义 ⊔S\sqcup{S}⊔S 为 SSS 的 最小上界 (least upper bound, lub, or join) ，对于 SSS 的每个上界 uuu ，满足 ⊔S⊑u\sqcup{S} \sqsubseteq u⊔S⊑u 。

  • 定义 ⊓S\sqcap{S}⊓S 为 SSS 的 最大下界 (greatest lower bound, glb, or meet) ，对于 SSS 的每个下界 lll ，满足 l⊑⊓Sl \sqsubseteq \sqcap{S}l⊑⊓S 。

若 ∣S∣=2|S| = 2∣S∣=2 ，例如 S={a,b}S = \{a,\ b\}S={a, b} ，则可记其 lub 为 ⊔S=a⊔b\sqcup{S} = a \sqcup b⊔S=a⊔b ，记其 glb 为 ⊓S=a⊓b\sqcap{S} = a \sqcap b⊓S=a⊓b 。

不是每个偏序集都有 lub 或 glb 。

如果偏序集存在 lub 或 glb ，则其 lub 或 glb 必定是唯一的。

  • 设 g1g_1g1​ 和 g2g_2g2​ 都是偏序集 PPP 的 glb ，则根据定义有 g1⊑g2∧g2⊑g1g_1 \sqsubseteq g_2 \land g_2 \sqsubseteq g_1g1​⊑g2​∧g2​⊑g1​ ，即 g1=g2g_1 = g_2g1​=g2​ 。

# 格

设 (P,⊑)(P,\ \sqsubseteq)(P, ⊑) 是一个偏序集，若对于 ∀a,b∈P\forall a,\ b \in P∀a, b∈P ，a⊔ba \sqcup ba⊔b 和 a⊓ba \sqcap ba⊓b 都存在，则称该偏序集为 格 (lattice) 。

设 (P,⊑)(P,\ \sqsubseteq)(P, ⊑) 是一个偏序集，若对于 ∀a,b∈P\forall a,\ b \in P∀a, b∈P ，a⊔ba \sqcup ba⊔b 存在或 a⊓ba \sqcap ba⊓b 存在，则称该偏序集为 半格 (semi-lattice) 。

  • 如果仅 a⊔ba \sqcup ba⊔b 存在，则称其为 join semi-lattice 。

  • 如果仅 a⊓ba \sqcap ba⊓b 存在，则称其为 meet semi-lattice 。

设 (P,⊑)(P,\ \sqsubseteq)(P, ⊑) 是一个格，若对于 ∀S⊆P\forall S \subseteq P∀S⊆P ，⊔S\sqcup S⊔S 和 ⊓S\sqcap S⊓S 都存在，则称该偏序集为 全格 (complete lattice) 。静态测试通常关注全格。

  • 所有的有限格都是全格，但全格不一定是有限格。

  • 对于全格，记其 top 为 ⊤=⊔P\top = \sqcup{P}⊤=⊔P ，bottom 为 ⊥=⊓P\bot = \sqcap{P}⊥=⊓P 。

几个有助于理解格的示例：

  • (Z,≤)(Z,\ \leq)(Z, ≤) 是格，其 ⊔\sqcup⊔ 和 ⊓\sqcap⊓ 可用 max⁡\maxmax 和 min⁡\minmin 表示。

  • (S,substring)(S,\ substring)(S, substring) 不一定是格，例如对于 {a,ab,ac,ad,abc}\{a, ab, ac, ad, abc\}{a,ab,ac,ad,abc} ，a⊔ada \sqcup ada⊔ad 不存在。

  • (P(S),⊆)(P(S),\ \subseteq)(P(S), ⊆) 是格，其 ⊔\sqcup⊔ 和 ⊓\sqcap⊓ 可用 ∪\cup∪ 和 ∩\cap∩ 表示。

  • (Z,≤)(Z,\ \leq)(Z, ≤) 不一定是全格，例如对于正整数的全集 Z+\bold{Z}^+Z+ ，⊔Z+\sqcup{\bold{Z}^+}⊔Z+ 不存在。

  • (P(S),⊆)(P(S),\ \subseteq)(P(S), ⊆) 是全格。

设 L1,L2,...,LnL_1, L_2, ..., L_nL1​,L2​,...,Ln​ 是 nnn 个格，如果每个格 Li=(Pi,⊑i)L_i = (P_i,\ \sqsubseteq_i)Li​=(Pi​, ⊑i​) 的 ⊔i\sqcup_i⊔i​ 和 ⊓i\sqcap_i⊓i​ 都存在，则定义这些格的 product lattice 为 Ln=(P,⊑)L^n = (P,\ \sqsubseteq)Ln=(P, ⊑) ，其中 P=P1×P2×...×PnP = P_1 \times P_2 \times ... \times P_nP=P1​×P2​×...×Pn​ ，并且满足如下性质：

  • (x1,x2,...,xn)⊑(y1,y2,...,yn)⇔(x1⊑y1)∧(x2⊑y2)∧...∧(xn⊑yn)(x_1, x_2, ..., x_n) \sqsubseteq (y_1, y_2, ..., y_n) \lrArr (x_1 \sqsubseteq y_1) \land (x_2 \sqsubseteq y_2) \land ... \land (x_n \sqsubseteq y_n)(x1​,x2​,...,xn​)⊑(y1​,y2​,...,yn​)⇔(x1​⊑y1​)∧(x2​⊑y2​)∧...∧(xn​⊑yn​)

  • (x1,x2,...,xn)⊔(y1,y2,...,yn)=(x1⊔y1,x2⊔y2,...,xn⊔yn)(x_1, x_2, ..., x_n) \sqcup (y_1, y_2, ..., y_n) = (x_1 \sqcup y_1,\ x_2 \sqcup y_2,\ ...,\ x_n \sqcup y_n)(x1​,x2​,...,xn​)⊔(y1​,y2​,...,yn​)=(x1​⊔y1​, x2​⊔y2​, ..., xn​⊔yn​)

  • (x1,x2,...,xn)⊓(y1,y2,...,yn)=(x1⊓y1,x2⊓y2,...,xn⊓yn)(x_1, x_2, ..., x_n) \sqcap (y_1, y_2, ..., y_n) = (x_1 \sqcap y_1,\ x_2 \sqcap y_2,\ ...,\ x_n \sqcap y_n)(x1​,x2​,...,xn​)⊓(y1​,y2​,...,yn​)=(x1​⊓y1​, x2​⊓y2​, ..., xn​⊓yn​)

如果构成 product 的每个格都是全格，则 product 也是全格。

设 LLL 是一个全格，定义 LLL 的 高度 (height) 为从其 ⊤\top⊤ 到 ⊥\bot⊥ 的最长路径的长度，记为 hLh_LhL​ 。

# 格与不动点

设 (L,⊑)(L,\ \sqsubseteq)(L, ⊑) 是一个格，对于函数 f:L→Lf : L \to Lf:L→L ，若 ∀x,y∈L,x⊑y⇒f(x)⊑f(y)\forall x,\ y \in L,\ x \sqsubseteq y \rArr f(x) \sqsubseteq f(y)∀x, y∈L, x⊑y⇒f(x)⊑f(y) ，则称 fff 是 单调的 (monotonic) 。

对于一个函数 fff ，若 X=f(X)X = f(X)X=f(X) ，则称 XXX 是 fff 的一个 不动点 (fixed point) ，记为 X=fFixX = f^{Fix}X=fFix 。

  • 函数的不动点可能不存在，也可能不唯一。

设 (L,⊑)(L,\ \sqsubseteq)(L, ⊑) 是一个全格，如果 LLL 是有限的，并且函数 f:L→Lf : L \to Lf:L→L 是单调的，则：

  • 不断迭代 f(⊤),f(f(⊤)),...,fk(⊤)f(\top), f(f(\top)), ..., f^k(\top)f(⊤),f(f(⊤)),...,fk(⊤) ，直至达到某个不动点，即为 fff 的 最大不动点 (greatest fixed point) 。

  • 不断迭代 f(⊥),f(f(⊥)),...,fk(⊥)f(\bot), f(f(\bot)), ..., f^k(\bot)f(⊥),f(f(⊥)),...,fk(⊥) ，直至达到某个不动点，即为 fff 的 最小不动点 (least fixed point) 。

  • fff 的不动点是存在的，并且 fff 的最大和最小不动点各是唯一的。

证明

以最小不动点为例，首先证明 fff 存在不动点：

由 f:L→Lf : L \to Lf:L→L 可知 f(⊥)∈Lf(\bot) \in Lf(⊥)∈L 。根据 bottom 的定义有 ⊥⊑f(⊥)\bot \sqsubseteq f(\bot)⊥⊑f(⊥) 。

因为 fff 是单调的，所以 f(⊥)⊑f(f(⊥))f(\bot) \sqsubseteq f(f(\bot))f(⊥)⊑f(f(⊥)) ，迭代地有 fi(⊥)⊑fi+1(⊥))(i∈N)f^i(\bot) \sqsubseteq f^{i + 1}(\bot))\ (i \in \bold{N})fi(⊥)⊑fi+1(⊥)) (i∈N) 。

因为 LLL 是有限的，所以 ∃H∈N\exists H \in \bold{N}∃H∈N 使得 fff 的值域落在 ⊥,f(⊥),f2(⊥),...,fH(⊥)\bot, f(\bot), f^2(\bot), ..., f^H(\bot)⊥,f(⊥),f2(⊥),...,fH(⊥) 之间。那么对于 i>Hi > Hi>H ，根据鸽巢原理有 ∃k≤H,fk(⊥)=fi(⊥)\exists k \le H,\ f^k(\bot) = f^i(\bot)∃k≤H, fk(⊥)=fi(⊥) 。

由 k<jk < jk<j 可知 fk(⊥)⊑...⊑fi(⊥)f^k(\bot) \sqsubseteq ... \sqsubseteq f^i(\bot)fk(⊥)⊑...⊑fi(⊥) ，即 fk(⊥)=fk+1(⊥)=...=fj(⊥)f^k(\bot) = f^{k + 1}(\bot) = ... = f^j(\bot)fk(⊥)=fk+1(⊥)=...=fj(⊥) ，因此 fff 存在不动点 fFix=fk(⊥)f^{Fix} = f^k(\bot)fFix=fk(⊥) 。

随后证明据此迭代得到的是 fff 的最小不动点：

不妨设 fff 存在另一不动点 x=f(x)x = f(x)x=f(x) 。根据 bottom 的定义有 ⊥⊑x\bot \sqsubseteq x⊥⊑x 。

因为 fff 是单调的，所以 f(⊥)⊑f(x)f(\bot) \sqsubseteq f(x)f(⊥)⊑f(x) ，迭代地有 ∀i∈N,fi(⊥)⊑fi(x)=x\forall i \in \bold{N},\ f^i(\bot) \sqsubseteq f^i(x) = x∀i∈N, fi(⊥)⊑fi(x)=x 。

由 iii 的任意性可知 fFix=fk(⊥)⊑xf^{Fix} = f^k(\bot) \sqsubseteq xfFix=fk(⊥)⊑x ，其中 kkk 是迭代达到第一个不动点时的幂数。由此可知迭代得到的是 fff 的最小不动点。

# 不动点理论与迭代算法

设 LLL 是一个有限的全格，我们可以令每个基本块上的值 vj∈Lv_j \in Lvj​∈L ，则其组合 (v1,v2,...,vk)(v_1, v_2, ..., v_k)(v1​,v2​,...,vk​) 是一个 product lattice LkL^kLk 。

随后，我们可以将每轮迭代 iteriiter_iiteri​ 等价为一个函数 F:Lk→LkF : L^k \to L^kF:Lk→Lk ，包括：

  • 每个基本块的 transfer function fi:L→Lf_i : L \to Lfi​:L→L 。

  • 表示数据流交汇的 join / meet function ⊔/⊓:Lm→L(m∈[2,k−1])\sqcup/\sqcap : L^m \to L\ (m \in [2,\ k - 1])⊔/⊓:Lm→L (m∈[2, k−1]) 。

可以证明 FFF 是单调的，因此可以证明数据流分析的迭代算法能够得到“最优解”，即其所到达的不动点是最大或最小不动点。

证明

在数据流分析中定义的 Gen/KillGen/KillGen/Kill 函数是单调的，因此我们只需证明 ⊔\sqcup⊔ 和 ⊓\sqcap⊓ 是单调的。

对于 ∀x,y,z∈L,x⊑y\forall x,\ y,\ z \in L,\ x \sqsubseteq y∀x, y, z∈L, x⊑y ，我们证明 x⊔z⊑y⊔zx \sqcup z \sqsubseteq y \sqcup zx⊔z⊑y⊔z ，即可证明 ⊔\sqcup⊔ 是单调的。

根据 ⊔\sqcup⊔ 的定义有 y⊑y⊔zy \sqsubseteq y \sqcup zy⊑y⊔z ，又 x⊑yx \sqsubseteq yx⊑y ，所以 x⊑y⊔zx \sqsubseteq y \sqcup zx⊑y⊔z 。由此可知 y⊔zy \sqcup zy⊔z 是 xxx 的一个上界。同时，y⊔zy \sqcup zy⊔z 也是 zzz 的一个上界。

而 x⊔zx \sqcup zx⊔z 是 xxx 和 zzz 的最小上界，所以 x⊔z⊑y⊔zx \sqcup z \sqsubseteq y \sqcup zx⊔z⊑y⊔z 。

设被分析的程序的 CFG 中有 kkk 个基本块，设与之对应的格 LLL 的高度为 hhh ，则数据流分析的迭代算法最多经过 khkhkh 轮迭代后停止。

  • 考虑最坏情况，假设每轮迭代中只有一个基本块上的格发生了一步变换（即只有一个 bit 0→10 \to 10→1）。

# 格与 May / Must Analysis

无论是 may analysis 还是 must analysis ，都是从 unsafe result 出发，向 safe result 前进，停留在二者之间的某个位置。

  • unsafe result 表示 no definitions can reach ，即什么都没有；

  • safe result 表示 all definitions may reach ，即什么都可能有；虽然是 safe 的，却是 useless 的。
